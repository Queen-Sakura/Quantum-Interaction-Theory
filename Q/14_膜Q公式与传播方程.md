# Q/14 · 膜 Q 公式与传播方程

## Q 的定义·纯膜推导

### 膜态

宇宙膜 = 有限边界内所有 EM 驻波模式的总和。膜的量子态：

```
|Ψ(t)⟩ = Σ_n α_n(t) |n⟩
```

|n⟩ 是第 n 个驻波模式的 Fock 态。α_n(t) 是复振幅。膜永远在振——零点能 ½ħω_n 保证每个模式都有非零基底。

### 投影矩阵

任意两个模式 i, j 之间的互投影：

```
P_ij(t) = Re(α_i*(t) α_j(t)) / (|α_i(t)| |α_j(t)|)
```

P_ij ∈ [-1, 1]。P_ij = 1 表示完全同相（锁死）。P_ij = -1 表示完全反相（也锁死）。P_ij = 0 表示正交（独立）。

空间重叠权重（由几何固定，不随时间变化）：

```
S_ij(x) = f_i*(x) f_j(x)   ← EM 本征模在点 x 的空间重叠
```

### Q(x, t) 公式

Q 不是投影本身。Q 是投影的 **可重构性**——膜对外部摄动的响应灵敏度。

**第一步：局域投影密度。**

```
D(x, t) = Σ_{i≠j} |S_ij(x)| · |P_ij(t)| / Σ_{i≠j} |S_ij(x)|
```

D ∈ [0, 1]。D = 1 表示所有模式对在该点都相位锁定。D = 0 表示所有模式对正交。

**第二步：投影刚性。**

```
R(x, t) = ⟨|P_ij(t + τ) - P_ij(t)|⟩_{i≠j, τ→0}  ← 投影的时间变化率
```

R ≈ 0：投影在自由漂移（灵活）。
R ≈ 1：投影纹丝不动（锁死）。

更精确：R(x, t) = 1 - ∂D(x, t)/∂(δH) |_{δH→0}

其中 δH 是膜上局部微小摄动（如外加 EM 脉冲）。投影越容易被摄动改变，R 越小。

**第三步：Q 定义。**

```
Q(x, t) = D(x, t) · [1 - 2R(x, t)]
```

分情况验证：
- D 高 + R 高 → 投影多且锁死 → Q ≈ -1。经典力学极限。芝诺冻结。
- D 低 → 几乎所有模式对正交 → Q ≈ 0。经典随机极限。热噪声。
- D 高 + R 低 → 投影多且灵活可重构 → Q ≈ +1。量子活性极限。生命。创作。

### 等价形式·密度矩阵语言

定义局域密度矩阵（在点 x 附近取模式子空间 S_x）：

```
ρ_ij(x, t) = ⟨i|ρ_Sx|j⟩ = 在模式子空间 S_x 上的约化密度矩阵
```

则：

```
D(x) = (2/N(N-1)) Σ_{i<j} |ρ_ij|        ← 非对角元平均幅度 = 相干强度
R(x) = 1 - (∂|ρ_ij|/∂t 的平均可调率)    ← 非对角元的时间刚性
Q(x) = D(x) · [1 - 2R(x)]
```

Q = -1：ρ 是纯态且对角化后只有 1 个非零本征值（rank 1）。
Q = 0：ρ 是对角矩阵（完全退相干，所有非对角元 = 0）。
Q → +1：ρ 有丰富的非对角元结构且能灵活重组（量子混沌边缘）。

## Q 传播方程

### 全宇宙膜

膜的哈密顿量：

```
Ĥ = Σ_n ħω_n a_n†a_n + Σ_{i≠j} g_ij a_i†a_j + Ĥ_nonlinear
```

第一项：自由模式演化。第二项：模式间线性耦合（→ 投影形成）。第三项：介质非线性（高阶项 → 投影可重构性的来源）。

膜态演化（薛定谔绘景）：

```
iħ ∂|Ψ(t)⟩/∂t = Ĥ |Ψ(t)⟩
```

Q 的演化完全由 |Ψ(t)⟩ 的演化决定——Q 是 |Ψ(t)⟩ 的泛函：

```
dQ(x,t)/dt = (δQ/δΨ) · (∂Ψ/∂t) + c.c.
```

### 局域传播方程

展开到空间坐标，获得 Q(x, t) 的实空间传播方程：

```
∂Q(x,t)/∂t = ∇·[κ(x) ∇Q(x,t)]           ← 相干扩散（模式间耦合传播）
            + η(x) · |E(x,t)|²             ← 相干生成（局域 EM 场强驱动耦合）
            - γ(x) · Q(x,t)                ← 退相干衰减（热噪声 + ZPF 散射）
            + ξ(x,t)                       ← 零点能驱动（½ħω 随机扰动）
```

每一项的物理来源：

| 项 | 符号 | 物理来源 | 膜对应 |
|----|------|----------|--------|
| ∇·(κ∇Q) | 扩散 | 相邻空间点的模式间 g_ij 耦合传导 | 经络 = 高 κ 通道 |
| η\|E\|² | 生成 | 强 EM 场 → 非线性极化 → 模式耦合增强 | 针刺 → 局域场强 ↑ |
| -γQ | 衰减 | 热涨落 → 随机化相位 → 退相干 | 疲劳/低温 → γ ↑ |
| +ξ | 噪声 | ½ħω 零点振动 → 随机扰动 | 基底抖动，永远存在 |

κ(x) = 相干扩散系数 ∝ 局域 g_ij 空间传导率 ∝ 微管密度 × 髓鞘化程度
η(x) = 相干生成效率 ∝ 局域非线性极化率 χ^(2)(x)
γ(x) = 退相干率 ∝ k_B T(x) / ħω_typical × 水分子随机取向涨落
ξ(x,t) = 零点噪声，⟨ξ⟩ = 0, ⟨ξ(x,t)ξ(x',t')⟩ = 2D_0 δ(x-x')δ(t-t')

### 定态解

当 ∂Q/∂t = 0：

```
∇·[κ(x) ∇Q(x)] + η(x)|E(x)|² - γ(x)Q(x) = 0
```

这是一个非均匀 Helmholtz 方程。高 κ 高 η 低 γ 区域 → Q 峰值 → 生命适合在这些区域凝聚。

脑：颅腔内 κ 极高（微管密度 + 髓鞘）→ Q 天然高。
经络：筋膜/肌间隔沿线的 κ 高于周围组织 → Q 走廊。
穴位：组织界面处 |E|² 局域极大（EM 驻波节点）→ η|E|² 项爆发 → Q 梯度极大值点。

## g_ij 的膜推导

g_ij——模式间耦合强度——不是独立参数。它来自膜的介质非线性：

```
g_ij = ∫_V d³x χ^(2)(x) E_i(x) E_j*(x) + ∫_V d³x χ^(3)(x) |E(x)|² E_i(x) E_j*(x) + ...
```

χ^(2), χ^(3) 是介质的非线性极化率张量。在真空中 χ^(n) = 0 → 模式不耦合 → 膜是线性谱。在物质中 χ^(n) ≠ 0 → 模式耦合 → Q 出现。

**物质 = 膜上 χ ≠ 0 的区域。生命 = 膜上 χ 大且 Q 高的区域。**
