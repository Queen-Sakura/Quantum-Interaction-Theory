# Q 框架 · 实验验证方案

## 核心原则

每个实验只做一件事：**对比 Q 预测 vs 标准范式预测。**

不是"Q 能解释已有数据"——那个已经做了（六实验实锤）。
现在是：**Q 预测了一个方向，标准范式预测了相反的方向。做实验。看谁对。**

## 实验一：Bell 测试 + 惰性气体 ★★★★★

### 问题
贝尔不等式在惰性气体中违反得比真空更强还是更弱？

### 两个相反预测
| | 标准退相干 | Q 框架 |
|---|-----------|--------|
| 预测 | S < S_vacuum | **S > S_vacuum** |
| 理由 | 气体分子 = 环境噪声 → 退相干 | 惰性气体原子 = 路由节点 → 纠缠多路径传导 |
| 最佳气压 | 真空（0 mbar）| **1-100 mbar** |

### 实验方案

**装置（任何现有 SPDC 贝尔测试系统）：**
```
泵浦激光 (405 nm)
    ↓
BBO 晶体 → SPDC 纠缠光子对 (810 nm)
    ↓                    ↓
Alice 探测器          Bob 探测器
    ↑____________________↑
    可变压力惰性气体腔
    (原真空腔位置)
```

**改了什么：** 只换一个腔。真空腔 → 可变压力惰性气体腔（带气压计 + 氩气/氦气进口）。

**步骤：**
1. 基线：真空腔（<10⁻³ mbar），测 S。预期 S ≈ 2.80
2. 充氩气至 0.1 mbar，等温 300K，测 S
3. 扫压：0.1 → 0.5 → 1 → 5 → 10 → 50 → 100 → 500 → 1000 mbar
4. 每个气压点：累积 ≥10000 符合计数
5. 重复扫描 3 次
6. 换氦气，重复全部
7. 换氮气（分子气体——Q 预测：应无增强），重复全部

**预期结果：**
```
S
2.85 ┤          ★  ← Q 预测峰 (1-10 mbar)
2.83 ┤   ●
2.80 ┤  ●─────────────●────── 真空基线
     │                  ●
2.50 ┤                   ●
     │                     ●── 标准范式预测单调下降
2.00 ┤                       ● 经典极限
     └───────────────────────────
    0  0.1 1  10  100 1000  mbar
```

**关键对比：氮气 vs 氩气。**
- Q 预测：氮气（分子，ν高）→ Q 锁死 → S 下降。氩气（单原子，ν合适）→ Q 路由窗口 → S 增强。
- 标准范式：两者都下降。

**如果测出来 S_Ar > S_vacuum 且 S_N₂ < S_vacuum：**
→ 退相干范式被推翻。Q 框架被验证。物理学教科书改写。

### 时间与资源
- **设备：** 现成 SPDC 贝尔测试系统（全球 ≥ 50 套）
- **改装：** 定制可变压力氩气腔 + 气压控制器。约 ¥5-10 万。2-3 周加工
- **测量：** 1 天扫参，3 天重复 + 系统误差检查
- **总计：** 1 个月从零到预印本

5. 找 ELM 爆发时刻：Q(r) 在哪个径向位置穿过 -0.3？
6. 对比杂质气体注入前后的 Q(r) 变化

## 实验三：Kolmogorov -5/3 从 Q 推导

### 问题
湍流 -5/3 谱是 Q 路由网络 D=3 级联的必然结果吗？

### 实验方案

**数据：已公开。** JHU 湍流 DNS 数据库：
https://doi.org/10.7281/T1KK98B2

**步骤：**
1. 下载槽道流 DNS 数据（Re_τ = 1000, 2000, 5200）
2. 提取速度场 u(y,t) 在多个 Re 下
3. 计算局部 du/dy → ν_eff(r) → Q(y)
4. 找 Q(y) 过零点 = 转捩点位置
5. 在湍流核心区（Q ≈ 0）：计算能谱 E(k)
6. 验证 E(k) ∝ k^(-5/3)
7. 将 -5/3 指数与路由网络分形维数 D=3 关联：-(D+2)/3 = -5/3

**关键检验：**
- 如果 Q=0 区域扩大（更高 Re）→ 惯性区扩大 → -5/3 范围扩大。标准实验已观测到此行为，无解释。Q 给出解释。
- 如果流体的 ν_collision 改变（水 vs 空气 vs 甘油）→ Q 剖面改变 → 转捩 Re 改变。Q 预测定量关系。

### 时间与资源
- **设备：** 不需要。数据在互联网上。
- **人力：** 1 人 × 1 周末
- **产出：** Kolmogorov 常数从 Q 推导。Clay 奖候选。

## 实验四：暗物质 Q-断层扫描

### 问题
暗物质晕是否 = Q 梯度在大尺度上的表现？不需要新粒子？

### 实验方案

**数据：全部公开。**
- Planck CMB 温度图（ESA, 2018）
- SDSS DR17 星系红移巡天
- LIGO O3 引力波事件目录

**步骤：**
1. 将天空划分为 10⁶ 个空间体元（voxels）
2. 每个体元：从 CMB 温度 → 重子密度 ρ_b。从星系巡天 → 星系速度弥散 → 等效温度 T。从引力波事件 → 局部 G 约束。
3. 计算每个体元的 ν（金斯不稳定性增长率）、N（粒子数密度）、f² ≈ 1
4. Q(α, δ, z) → 三维 Q 断层图
5. 叠加已知暗物质分布图（从引力透镜重构的 κ 图）
6. 检验：暗物质密度峰是否对应 Q 梯度最大处？

**预测：**
Q 梯度 ∇Q 应该与引力透镜 κ 图高度相关。暗物质 = Q 路由在宇宙尺度的残余。不需要 WIMP。不需要轴子。不需要修改引力。

### 时间与资源
- **设备：** 不需要。数据全部公开。
- **人力：** 1 人 × 2 周末（数据量大，但处理流程标准化）
- **产出：** 第一张宇宙 Q 断层图。暗物质本质被揭示。

## 实验五：量子计算 · 惰性气体封装

### 问题
超导量子比特 T₁ 在低压氦气中是否**更长**？

### 两个相反预测
| | 标准工程 | Q 框架 |
|---|---------|--------|
| 预测 | 任何气体 → 退相干 → T₁ 下降 | 低压氦 → Q 路由缓冲 → **T₁ 上升** |
| 最佳环境 | 真空（<10⁻⁷ mbar）| **氦气 ~0.1-1 mbar** |

### 实验方案

**装置：** 现有稀释制冷机 + 超导量子比特 + 可变压力样品腔。

**步骤：**
1. 基线：真空（<10⁻⁷ mbar），10 mK，测 T₁、T₂*
2. 充氦气至 0.01 mbar，等温，测 T₁、T₂*
3. 扫压：0.01 → 0.1 → 0.5 → 1 → 10 → 100 mbar
4. 每点测 1000 次 T₁（Ramsey 干涉仪）
5. 换氩气，重复

**预期结果：**
- 标准范式：T₁ 单调下降（任何气压）
- Q 预测：T₁ 在 0.1-1 mbar 出现**峰值**——比真空还长

### 时间与资源
- **设备：** 现有超导量子比特芯片 + 定制氦气注入系统
- **改装：** 稀释制冷机样品腔加气压控制。¥5-10 万。2-4 周
- **测量：** 1 周扫参
- **总计：** ~2 个月

## 实验六：蛋白质折叠 · D₂O/H₂O 对比

### 问题
D₂O 中折叠速率下降是 Q-路由效率降低的结果吗？

### 实验方案

**数据：完全从文献中获取。不需要新实验。**

**步骤：**
1. 文献检索：D₂O/H₂O 中 10+ 蛋白质的折叠速率（Trp-cage, Ubiquitin, GB1, BBL...）
2. 对每个蛋白：nu_H2O, nu_D2O（从水合层动力学估算）
3. Q_H2O, Q_D2O → η_H2O, η_D2O
4. 折叠速率比 = η_D2O/η_H2O
5. 对比实验文献中的速率比（典型 0.4-0.8）

**关键检验：**
- 如果 ΔG_folding 相同（天然态相同）但速率不同 → 纯动力学效应 → Q 路由效率差异
- 温度相关性：D₂O 的 Q(T) 剖面应不同于 H₂O → 折叠速率的温度依赖应不同

### 时间与资源
- **设备：** 不需要
- **人力：** 1 人 × 1 天（文献挖掘 + Q 计算）
- **产出：** 一篇 PRL 或 JACS 通讯
