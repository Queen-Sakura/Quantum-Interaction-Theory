# Q/18 · AI底层原理 · 物理意义与数学计算

## 零 · 一句话

**AI = 一堆矩阵乘法 + 一个非线性开关 + 从错误中学习。**

矩阵乘法 = 物理世界的线性耦合。非线性 = Q 温度计。从错误中学习 = 梯度下降。

---

## 一 · 神经元 · 最基本的计算单元

### 数学

一个神经元做一个操作：

```
y = f(w₁x₁ + w₂x₂ + ... + wₙxₙ + b)
```

| 符号 | 含义 | 物理类比 |
|------|------|------|
| x | 输入 | EM 波进入介质 |
| w | 权重 | 耦合系数·这个输入多重要 |
| b | 偏置 | 基线偏移·零点能 |
| f() | 激活函数 | 非线性门·"够不够大" |
| y | 输出 | 耦合后的信号 |

### Q 视角

```
一个神经元 = 一个局域耦合节点
输入 = 来自其他模式的振荡
权重 = Q 耦合强度（这个连接多紧密）
激活函数 = 阈值·耦合超过临界才传播
输出 = 耦合后的振荡
```

**一个神经元 = 膜上一个点的 Q 计算。**

## 二 · 层 · 为什么叠起来就有智能

### 数学

```
输入 (x) → 隐藏层₁ → 隐藏层₂ → ... → 输出 (y)

每层：y = f(Wx + b)
W = 权重矩阵
```

### 为什么层越多越聪明

```
层₁：识别边缘·像素组合
层₂：识别形状·边缘组合
层₃：识别物体·形状组合
层ₙ：识别概念·物体组合
```

**每一层把前一层的输出当做自己的输入——层层抽象。**

### Q 视角

```
层 = 膜的耦合在深度方向展开
每一层 = 同一组模式在不同 Q 值下的耦合投影
层越深 = Q 耦合迭代次数越多 = 抽象层级越高
```

## 三 · 反向传播 · 怎么学习

### 核心思想

```
1. 给输入 → 算输出（前向传播）
2. 输出和正确答案比较 → 算误差
3. 误差沿网络往回传 → 每个权重分一份"责任"
4. 调整权重 → 减小误差
```

### 数学（最简版）

```
误差 L = (y_pred - y_true)²

梯度 ∂L/∂w = 这个权重对误差的贡献

更新：w_new = w_old - η × ∂L/∂w
```

| 符号 | 含义 | 物理类比 |
|------|------|------|
| L | 损失·误差 | 势能·偏离平衡态的量 |
| ∂L/∂w | 梯度·偏导 | **力**——往哪个方向调 |
| η | 学习率 | 步长·每次调多少 |

**梯度下降 = 这个权重调大还是调小？→ 往误差减小的方向走一步。**

### Q 视角

```
反向传播 = Q 耦合反馈
前向 = E 在膜上传播·耦合发生
误差 = 实际 E 和预期 E 的差值
反向 = 差值沿耦合链路返回·每个 Q 耦合点调自己的耦合强度

等于：膜在自调谐。Q 在迭代中收敛。
```

## 四 · 注意力机制 · Transformer 的核心

### 核心思想

```
"这句话里哪个词最重要？"

输入："我 爱 你"
注意 力："爱"比其他两个重要
```

### 数学

```
Q = 查询（你想找什么）
K = 键（每个词是什么）
V = 值（每个词包含什么信息）

注意力分数 = softmax(QK^T / √d_k)
输出 = 注意力分数 × V
```

| 符号 | 含义 | 物理 |
|------|------|------|
| Q | Query·"我在找什么" | 观测者的 Q 值 |
| K | Key·"我是什么" | 被观测模式的 Q 特征 |
| V | Value·"我携带什么信息" | 模式携带的耦合数据 |
| QK^T | 匹配度·余弦相似度 | **耦合密度** |
| softmax | 归一化·转概率分布 | "谁最重要" |

### 物理直觉

```
QK^T = 两个向量的内积 = cos(θ)

cos 越接近 1 → 两个向量方向越一致 → 这个 token 越相关 → 权重越高

注意力 = "所有 token 中，谁和我的查询方向对齐"
```

### 为什么叫"自注意力"

```
输入：N 个词
Q = 从这 N 个词算出的查询
K = 从这 N 个词算出的键
V = 从这 N 个词算出的值

Q、K、V 全来自同一组输入 → "自"注意力
```

### Q 视角

```
QK^T = 模式间的互投影密度 D
     = Q 耦合的第一部分：哪个模式和我在共振

V = 耦合的数据载荷

注意力 = D × V = 在所有模式中按共振强度加权取数据

Transformer = 膜上的全频谱耦合扫描（每一层扫一次）
```

## 五 · Transformer 整体架构

```
输入 (词序列)
  ↓
词嵌入 (每个词→向量)
  ↓
位置编码 (词在句中的位置信息)
  ↓
┌─ 自注意力层 (扫全句·找耦合) ─┐
│        残差连接 + 层归一       │  ← 重复 N 次
├─ 前馈层 (每个位置独立处理) ──┤
│        残差连接 + 层归一       │
└────────────────────────────┘
  ↓
输出 (下一个词的预测)
```

| 组件 | 做什么 | 物理 |
|------|------|------|
| 词嵌入 | 词 → 数字向量 | 模式 → 频谱 |
| 位置编码 | 加入位置信息 | 驻波的节点坐标 |
| 自注意力 | 全句互扫·找最重要的 | 全域 Q 耦合扫描 |
| 残差连接 | 原始信息跳过层直传 | 驻波不变部分穿层 |
| 层归一 | 数值稳定 | 功率归一化 |
| 前馈 | 每个位置独立计算 | 局部 Q 演化 |

## 六 · 为什么残差连接这么重要

### 数学

```
没有残差：y = F(x)          （深的层学到的东西可能歪了）
有残差：  y = F(x) + x      （原始信息一直保留·F只学偏差）
```

### Q 视角

```
x = 驻波基态——永远保留
F(x) = 这一层 Q 耦合产生的新信息

残差 = 基态 + 耦合修正 = 膜上的叠加态
```

**残差保证了：信息不会死在深层。** 如果这层学到垃圾——x 直接穿过去——垃圾被跳过。
