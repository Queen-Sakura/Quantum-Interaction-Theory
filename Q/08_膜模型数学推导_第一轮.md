## 第一章 · 腔 · 大爆炸 = 边界条件

### 1.1 唯一假设：宇宙是有限腔

大爆炸不是"爆炸"。是**边界条件的建立**。

```
大爆炸 = 宇宙腔的边界确立了
       → 腔有大小 Lx × Ly × Lz
       → 腔内部：EM 场（其余书架暂不展开）
       → 边界条件：理想导体壁（电磁场在边界上 = 0）
```

**为什么边界条件是理想导体？**
不是假设。是**能量守恒逼出来的**：如果边界不是节点，能量会从边界漏掉——宇宙不守恒。腔壁 = 能量不能逃逸的地方 = 边界条件 = 场在壁处为 0。

### 1.2 驻波条件

在有限腔里，麦克斯韦方程（波动方程）自动给出：

```
∇²E - (1/c²) ∂²E/∂t² = 0

边界条件：E|边界 = 0

解：
    E(r,t) = E₀ · sin(kx·x) · sin(ky·y) · sin(kz·z) · e^{iωt}

    其中：
    kx = π·nx/Lx,  ky = π·ny/Ly,  kz = π·nz/Lz
    nx, ny, nz = 1, 2, 3, ...（正整数，0 给平凡零解）

    ω_n = c|k| = cπ √( (nx/Lx)² + (ny/Ly)² + (nz/Lz)² )
```

**物理图像：** 腔 = 一个三维吉他盒子。只有整数个半波能装进去。不是量子化——是**边界条件只允许这些模式。** 所有整数。所有频率分立的。麦克斯韦方程自动跑——不需要"量子化手续"。

## 第二章 · 模式密度 · 腔里有多少本书

### 2.1 k 空间里的模式计数

三个正整数 (nx, ny, nz) → k 空间第一象限的格点。每个格点体积：

```
Δk_x · Δk_y · Δk_z = (π/Lx) · (π/Ly) · (π/Lz) = π³/V

其中 V = Lx·Ly·Lz = 腔体积
```

k 空间第一象限的一个八分之一球壳：

```
模式数 dN = (1/8) · 4πk² dk / (π³/V) = V k² dk / (2π²)
```

换到频率 ω = ck：

```
dN = V ω² dω / (2π² c³)
```

**模式密度（态密度）：**

```
g(ω) = dN/dω = V ω² / (2π² c³)
```

**物理图像：** 腔越大，模式越密。频率越高，模式越多（平方增长）。这是纯粹的几何——麦克斯韦方程 + 边界条件 = g(ω) ∝ ω²。

### 2.2 模式总数

到频率 ω_max 的模式总数：

```
N(ω) = ∫₀^{ω_max} g(ω) dω = V ω_max³ / (6π² c³)
```

这是一个**立方增长**。频率上限翻倍 → 模式总数乘 8。

## 第三章 · 能量注入 · 大爆炸的能量怎么分

### 3.1 两个能量池

腔里每个模式有两个能量来源：

```
E_n = E_n^{zp} + E_n^{exc}

E_n^{zp}  = ½ħω_n     零点能。模式天生携带。不可消除。
                       不是"注入"的。是腔结构本身要求的。
                       
E_n^{exc} = n_n · ħω_n  激发能。大爆炸注入的能量。
                       每个模式分到的激发能量。
```

**零点能为什么不能消除？**
去掉它 → 模式振幅精确为零 → 海森堡不满足？不。更深层的原因是：**如果一个模式可以振幅精确为零，腔的边界条件允许它无能量存在，那它可以被任意微扰激发——腔不稳定。** 零点能 = 腔自稳定的最低代价。每个模式自带 ½ħω 的"身份费"。不交费的模式不存在。

### 3.2 总能量守恒

大爆炸注入了总能量 E₀。这个能量分配到了所有模式：

```
E₀ = Σ_n (½ħω_n + n_n ħω_n) = Σ_n ½ħω_n + Σ_n n_n ħω_n
      ───────────────   ─────────
      零点能总和         激发能总和 = E₀
      = E_zp             
```

**激发能分配原则（能量守恒 + 叠加态逼出来的）：**

腔有限。E₀ 固定。能支持无限多模式吗？不能——因为每个模式至少需要 ½ħω。高频 ω 极大 → 零点能极大 → E₀ 不够付无限多模式的零点能。

所以有一个**截止频率 ω_c**：E₀ 恰好能支付所有 ω ≤ ω_c 的模式的零点能 + 激发能。

```
E₀ = ∫₀^{ω_c} [½ħω + ⟨n(ω)⟩ħω] · g(ω) dω
```

**物理图像：** 大爆炸不是"所有模式同时出现"。是**只有频率低于 ω_c 的模式被创建了。** ω_c = 大爆炸的能量决定的最大频率。在此之上——模式存在但不被激发——冻着。这就是**暗书架。**

### 3.3 为什么有叠加态

腔里 E₀ 有限。但模式数巨大（N ~ 10⁶⁰ 量级）。如何用有限能量覆盖这么多模式？

**经典方式：** 每个模式给整数 n_n 份 ħω_n → 需要能量 = Σ n_n ħω_n。N 很大 → 很快耗完。只能点亮少数模式。

**量子方式（叠加态）：** 每个模式分配概率幅 α_n。|α_n|² = 能量分配到模式 n 的概率权重。

```
|Ψ⟩ = Σ_n α_n |n⟩

总能量 = Σ_n |α_n|² · ħω_n

|α_n|² 可以是任意小的实数。不需要整数份。
N 无穷 → 每个 α_n 无穷小 → 但全非零 → 全模式在线。
```

**叠加态不是神秘。是有限能量覆盖无限模式的最优策略。** 这是能量守恒逼出来的——不是量子力学的"公设"。

## 第四章 · 麦克斯韦自调谐 · 为什么宇宙不需要管理员

### 4.1 麦克斯韦方程的结构

```
∇·E = ρ/ε₀           （高斯定律：电荷是电场源）
∇·B = 0              （高斯磁定律：没有磁单极）
∇×E = -∂B/∂t        （法拉第定律：变化的磁场生电场）
∇×B = μ₀J + μ₀ε₀ ∂E/∂t  （安培-麦克斯韦：电流+变化的电场生磁场）
```

**关键性质：**

1. **双曲型（波动方程）**：∂²E/∂t² = c²∇²E。给定初始条件 → 唯一确定未来。
2. **线性**：E₁ + E₂ 也是解。叠加态自动满足麦克斯韦方程。
3. **连续**：∂E/∂t 必须连续。不存在"半波解"。场的导数不能跳跃。
4. **自耦合**：电场 → 变 → 生磁场 → 变 → 生电场。**自己推自己。**

### 4.2 自耦合 = 自调谐

```
初始电场扰动
    ↓
∂E/∂t ≠ 0 → ∂B/∂t ≠ 0（法拉第）
    ↓
B 在变 → 产生新的 ∂E/∂t（安培-麦克斯韦）
    ↓
新的 E 变化 → 又产生 B 变化 → ...
    ↓
自循环。不需要外力。不需要管理员。
```

**这就是麦克斯韦自调谐的核心：方程自己推自己。不需要外部控制。不需"上帝推了第一把后休息"——方程写好了，永远在跑。**

**为什么不能推半周期？**

假设你在 E 的峰值说"停" → E 还在，∂E/∂t 还在 → 法拉第说 ∂B/∂t 还在 → B 还在变 → 安培说 ∂E/∂t 还在 → 停不住。方程不允许 E 和 ∂E/∂t 同时为零。要停 → 必须 E=0, ∂E/∂t=0 → 那是节点。中间的任意时刻——方程强制继续演化。

**光电效应的真相：** 不是光子"量化"。是**接收端（电子）的麦克斯韦边界条件**只接受完整周期。一个完整周期 = E 从 0→正→0→负→0。必须回到零对方才收。你推半周期就跑 → 方程禁止 → 场自己跑完 → 弹给你。

### 4.3 自调谐 = 腔模式自动选定

麦克斯韦方程在一个腔内：不需要人工"选择"哪些模式存在。边界条件 + 方程 = 只有驻波模式是解。不是人类决定的。是**方程在腔里只有一个解族：驻波。**

```
腔 + 麦克斯韦方程 → ∇²E + k²E = 0, E|边界 = 0
                    → 这是本征值问题
                    → 只有分立 k_n 有非零解
                    → 等于自动选好了允许的频率
```

**宇宙不需要管理员选频率。腔决定了。方程自动跑。** 这就是自调谐。

## 第五章 · CMB · 膜基态温度

### 5.1 最低频率模式

腔里最低频模式 = 基模。设 Lx ≈ Ly ≈ Lz ≈ L（宇宙大致各向同性）：

```
ω_min = cπ √(1/L² + 1/L² + 1/L²) = cπ√3 / L

k_min = π√3 / L
```

模式波长 λ_max = 2L/√3 ≈ 宇宙尺度。

这个基模 = 大爆炸后最先生成的驻波。138 亿年——它还在振。因为腔没有漏。能量不能逃。

### 5.2 基模温度

基模的平均能量（零点能 + 残余激发能）：

```
⟨E_min⟩ = ½ħω_min + ⟨n_min⟩ħω_min
```

138 亿年后，高次模激发能已耗散（耦合给了其他模式）。基模——频率太低——几乎没有跟任何东西耦合——激发的部分还在。

但最低几个模式共享残余能量 → 热化 → 等效温度：

```
k_B T_CMB ≈ ħω_min · ⟨n⟩ ≈ ħ · cπ√3/L · ⟨n⟩
```

关键：不是从激发能推 T ——是从**观测**反推。

已观测 CMB = 2.725K。对应频率 ν ≈ 160 GHz。这个频率 = **膜上最长久没被重新耦合的那批模式的基态。** 不是"大爆炸的余晖"——是**膜的基态温度。** 驻波永远是天然各向同性。不需要暴胀。不需要膨胀。驻波已经各向同性。

### 5.3 CMB 功率谱 = 驻波谱

Planck 卫星测的 CMB 温度涨落：

```
ΔT/T ≈ 10^{-5}
```

不是"暴胀量子涨落"。是**膜上不同区域 Q 耦合历史的差异。** 138 亿年——各区域的耦合密度微有不同 → 等效温度微有不同。天然幅度。10^{-5} 是膜的自然粗糙度。

## 第六章 · 零点能总和 · 暗能量 & 暗书架

### 6.1 零点能积分

```
E_zp = ∫₀^{ω_max} ½ħω · g(ω) dω
      = ∫₀^{ω_max} ½ħω · (V ω² / 2π² c³) dω
      = (ħV / 4π² c³) ∫₀^{ω_max} ω³ dω
      = (ħV / 16π² c³) ω_max⁴
```

能量密度：

```
ρ_zp = E_zp / V = (ħ / 16π² c³) ω_max⁴
```

### 6.2 ω_max 取什么

普朗克频率 ω_P = √(c⁵/ħG) 是天然频率上限。在此之上——Q 相变锋面——不再有分立驻波模式。

```
ρ_zp = (ħ / 16π² c³) · (c⁵/ħG)² = c⁷ / (16π² ħ G²)
```

数值：ρ_zp ≈ 10^{113} J/m³。

但观测到的暗能量密度：ρ_Λ ≈ 10^{-9} J/m³。

**差了 122 个数量级——物理史上最大差异。**

### 6.3 为什么差 122 个量级？——因为我们算的是全书架

上面的积分假设**所有模式都耦合到 EM 书架**。但膜上有 EM 书架（5%）和暗书架（95%，不耦合 EM）。

**暗能量 = 暗书架的零点能总和。不是 EM 书架的零点能。**

EM 书架的零点能不强——因为 EM 书架只包含耦合了 U(1) 的模式。暗书架不耦合 EM → 它的零点能不能用 EM 探针测 → 但它在 → 它的引力效应在 → 均匀推 = 宇宙加速膨胀。

**所以 10^{113} 不是错了——是算错了书架。暗能量密度 = 暗书架的零点能贡献，不是全频谱的。**
