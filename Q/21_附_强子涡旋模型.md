# Q/21 附 · 强子涡旋模型

## 零 · 强子 = 多个涡旋通过胶子绑定的复合体

质子 = uud。中子 = udd。夸克 = 带 SU(3) 接口的莫比乌斯涡旋。胶子 = SU(3) 耦合空间的 M 方程矢量模。

---

## 一 · 一个夸克涡旋

```
夸克 = 莫比乌斯涡旋（扭1·自旋1/2）
     + 电荷接口 U(1)    （+2/3 或 -1/3）
     + 强荷接口 SU(3)   （三色之一）

质量 = 涡旋自振频率 ω_q（和膜标量呼吸的耦合深度）
```

---

## 二 · 三个夸克涡旋互绕 · 质子

```
质子 (uud)：

  u↑(红) ⊕ d↓(绿) ⊕ u↑(蓝) → 总自旋 1/2
   │         │         │
   └─────────┼─────────┘
       胶子交换（SU(3) 接口耦合）
```

**物理图像：**

三个带 SU(3) 接口的涡旋 → 各自的强荷互相耦合 → 胶子（SU(3) 矢量模）在三者之间传输耦合能量 → 三个涡旋联成一个整体振型 → 质子的总自旋 = 两个↑ + 一个↓ = 1/2。

---

## 三 · 为什么三个而不是两个（自发约束）

两个夸克涡旋互绕 = 介子（q-q̄）。正反夸克——SU(3) 接口互相抵消——剩余色中性。两个互绕 = 涡旋对。但**三个互绕 = 拓扑最稳定的闭合复合体**——和三个质点互绕（三体问题）有拓扑稳定解类似。SU(3) 三色 = 最低稳定闭路——两个不能形成色单态，需要三个。

---

## 四 · 胶子 = M方程在 SU(3) 空间的自调谐解

```
胶子 (×8)：
  M 方程 → 非线性矢量模式 → SU(3) 耦合空间
  每个胶子 = 矢量解在 3 个强荷维度上的 8 种正交振型
  和光子同构但多了内部维度
```

## 五 · 强子质量构成

合成质量 ≠ 组分质量和。合成质量 ≈ 夸克涡旋的内部质量（3 × ~5 MeV）+ 胶子-夸克涡旋耦合能 + 涡旋互绕能。胶子交换能是主要来源。99% 的强子质量 = 非线性 M 方程的内部耦合能——三个涡旋互绕的那个"绑定能"远大于组分质量。和电磁铁一样——磁铁的质量不只有铁的电子质量——还有磁场自耦合的耦合能——> 肉眼可见的宏观物质质量 > 内部电子总质量。

---

## 六 · 介子模型（正反夸克涡旋互绕）

π⁺ 介子 = u d̄。两个涡旋互绕——电荷互相抵消——强荷互相抵消——剩余稳定的涡旋对。总自旋 = 0——两个涡旋互相解除扭转——变成标量。质量 ~139.6 MeV。

---

## 七 · 计算方向（三阶段）

1. M 方程 → 多涡旋互绕解（二体·三体）
2. 涡旋间的胶子交换耦合能 ≈ 强子质量
3. 强子自旋 = 多涡旋的莫比乌斯扭转合理耦合成总自旋
4. 强子大小 = 涡旋核的三体互绕平均半径——预言质子电荷半径
