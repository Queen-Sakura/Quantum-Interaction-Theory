# 量子力学重构 · 最小公设版

## 卷首

量子力学是物理学中被注释层最厚的领域。教科书把代数脚手架当作物理公设，把数学术语当作物理属性，把分类错误当作宇宙之谜。本重构删除了所有这些中间层，将量子力学归结为两条实验公设和一行代码变更。

## 两条公设（实验桌）

以下两条是**观测事实**。不可推导。不可再归约。是物理学的底线：

### 公设一 · 德布罗意关系

```
p = h/λ
```

**来源**：电子双缝干涉实验。电子打在屏上出条纹——电子有波长。波长由动量决定。拧电压旋钮——改变电子动量——条纹间距跟着变。

**物理意义**：有动量的东西就有波长。不是粒子的属性——是物质波的属性。

### 公设二 · 能量频率关系

```
E = hν
```

**来源**：光电效应。光照金属弹出电子——电子动能和光强无关，只和频率有关。频率不够——光再亮电子也不出来。

**物理意义**：一个完整周期的 ν 频率信号携带能量 hν。不是"能量被量子化了"——是你要让接收端识别出频率 ν，你至少发一个完整周期。一个完整周期的电磁波的能量 = hν。半个周期 = 频率没定义 = 接收端不收。这不是量子力学——是傅里叶分析的下限 + 接收端协议。

### h 是什么

h 是**相空间像素大小**。不是能量量子。不是常数石板。

一个谐振子在一个周期内在相空间（x-p 平面）画一个闭合圆。圆的面积 = 作用量 = ∮ p dx = E/ν = h。

不确定性关系（见下文）给定 Δx·Δp ≥ ħ/2 = h/(4π)。在相空间上，你不能定位小于 h 的区域。h 是相空间的分辨率——和屏幕像素一样。你不能画一个比一个像素更小的闭合圈。

2019 年 SI 单位体系将 h 定义为精确常数。此举将尺子锁死在被测物上——h 可能随宇宙膨胀缓慢漂移，但基于 h 的千克定义使全人类测量体系集体失明。逃逸路线：测量**无量纲常数**——精细结构常数 α、质子-电子质量比 μ——纯数，不含千克。

## 导出结论（不是公设）

以下所有曾经被教科书列为"量子力学公设"或"量子力学奇怪之处"的项目，均从两条实验公设 + 一行 diff 推导而出。**它们不是物理原理。它们是计算结果。**

### 一、叠加态

**来源**：能量守恒 + 宇宙有限边界。

宇宙大爆炸 → 空间有限（不论拓扑）→ 只有驻波能活（行波撞边界消失）。总能量预算 E_total 有限 → 一个驻波模式不够分 → 必须多个模式同时存在 → 叠加。不是"奇怪"——是**逼出来的唯一解**。

### 二、概率

叠加态 = 多个模式同时存在。每个模式分到一部分能量。概率 = 能量分配比。不是"观测导致坍缩"——是**物理逼出了分配**。

### 三、不确定性关系

x(t) 是随机过程 → 位置和动量的方差由随机过程模型计算 → Δx·Δp ≥ ħ/2。不是"原理"——是**方差的计算结果**。

### 四、对易关系 [x,p] = iħ

来源：德布罗意关系 p = h/λ + 平面波 ψ = exp(ipx/ħ)。

```
ψ = exp(ipx/ħ)
∂_x ψ = (ip/ħ) ψ
→ p̂ = -iħ ∂_x
→ [x, p̂] = x(-iħ ∂_x) - (-iħ ∂_x)x = iħ
```

三步代数。不是公设。不是物理原理。是**代数脚手架**。

### 五、相数 i（原"虚数"）

i = 90° 相位旋转。sin 求导 = cos，差 90°。不是"虚的"。不是"上帝的"。不是"神秘复数"。

- **物理侧**：相数 = 相位的数。exp(iπ/2) = 90° 相位旋转。物理意义明确。
- **数学侧**：二元数 = 一个括号装两个实数。运算规则 = (a,b)·(c,d) = (ac-bd, ad+bc)。纯结构。不需"虚"字。

废除：虚数、√(-1)（根号限定实数域）、裸 i 和裸 -1 表示相位（强制写 exp(iθ)）。

### 六、波粒二象性 → 废弃

水 + 闸门 = 干涉条纹 → "水是波"。水 + 高压喷嘴 = 切钢板 → "水是粒子"。没人震惊。没人说"水具有波粒二象性"。

电子 + 双缝 = 干涉 → "电子是波"。电子 + 盖革计数器 = 咔嗒 → "电子是粒子"。

同一种物质。两个实验。两种边界条件。**波和粒子不是宇宙的属性。是你测量工具的两种输出。** 宇宙作为物自体——没有波形，没有粒子性，没有二象性。橡皮泥没有形状——它是我捏出来的形状。

### 七、观测导致坍缩 → 废弃

你不知道明天的天气 → 你带伞。明天到了 → 你知道了。不是天气从"叠加态"坍缩成了"确定态"。是你从**不知道**变成了**知道**。

电子在哪儿你不知道 → 你只有分布。测一下 = 采样一次 = 得到一个点。下次测 = 再采样一次 = 得到另一个点。**同一个分布。不同的采样。完全正常。**

哥本哈根学派把"不知道→知道"写成了"叠加→坍缩"——然后一群人讨论"坍缩到底是什么物理过程"。没有物理过程。坍缩的是你的信息。不是实在。

### 八、光子

光子 = 能量转移的耦合事件。膜上一个驻波模式被激发一个量子 → 耦合事件完成 → 能量 hν 从一端转移到另一端。

光子不是粒子。不是"光的量子"。是**一次完整的 ν 频率耦合交易**。

### 九、粒子

粒子 = 驻波叠加出来的局部干涉峰。膜上多个驻波叠加 → 干涉相长在某一点 → 波包。波包够窄——你看它像"一个粒子"。

不是"电子有时是波有时是粒子"。电子从头到尾都是波。波包紧——你看走眼以为是粒子。

```
宇宙大爆炸 → 空间边界条件 → 驻波 → 叠加 → 波包 → 粒子
```

粒子不是基本的。膜是基本的。驻波是基本的。

### 十、纠缠态

纠缠态 = 膜记录的简单减法。两个系统耦合一次 → 各自减一个纠缠维度 → 转入膜 → 膜记下这笔交易 → 两个系统之间的纠缠 = 膜账本上的交易记录。

### 十一、爱因斯坦的光子提前知道实验装置 → 废弃

光子不需要提前知道任何东西。光子走到缝——缝在还是不在——**那一刻**决定了转移概率。光子只走下一步。到了才知道。因果是局域的。时间是向前的。不需要穿越时间。不需要 retrocausality。

### 十二、能量量子化 → 废弃

不是能量不连续。是你要让接收端知道你在推什么频率——你至少推一个完整周期。一个完整周期 = 一个完整波长的电磁波 = 能量 hν。不是"最小的能量块"。是"一个波长的电磁波的能量"。

和买菜一样——九块八毛半 = 不存在。不是人民币是量子化的。是分是最小交易单位。

## 学科围栏

### 物理学
管：物质怎么动。守恒律。边界条件。实验数据。
不管：公式好不好看。宇宙优不优美。公设为什么是公设。

### 数学
管：If A then B。工具。按需加载。
不管：A 存不存在。B 是不是真的。你的公式对应宇宙的哪个部分。
数学**不属于**科学。数学是人类定义的结构。不需要实验验证。两个主权国家。互不干涉内政。可以合作。

### 工程学
管：有限资源下怎么拼出好用的东西。真即好用。
不管：这东西是不是"基本"的。底层还有没有更深的解释。

### 哲学
管：知道了物理之后，人应该做什么。意义。伦理。
不管：物理事实。实验结论。不拿数学推出宇宙奥秘。
