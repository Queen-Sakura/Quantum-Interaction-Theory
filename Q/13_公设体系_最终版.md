**陈述**：
```

∇×B = μ₀ε₀ ∂E/∂t
```

**作用**：决定膜上有哪些驻波模式。决定全宇宙 EM 场的运动学。

### 第二公设：薛定谔方程

**陈述**：
```
iħ ∂ψ/∂t = Ĥψ
```

**作用**：决定一个激发如何在膜上的多个模式之间分布。决定叠加和概率幅。

**叠加的来源**：
- 麦克斯韦给出离散模式空间（舞台）
- 薛定谔给出激发跨模式分布的规则（演员走位）
- 叠加 = 同一个激发同时占用多个模式子空间
- 测量塌缩 = 测量装置的有限耦合带宽只接入其中一个模式——其他模式的系数对装置不可见

**覆盖的旧概念**：
- 对易关系 → 被降级。三步代数推导。不是公设。
- 概率诠释 → |ψ|² = 概率 = 测量耦合的部分性。不是 bug——是生命宇宙的门。
- 观测坍缩 → 未知→已知。采样。不是物理过程。
- 波粒二象性 → 废弃。边界条件决定观测结果。橡皮泥无形状。
- 虚数 → 相数。√(-1) 废弃。
- 能量量子化 → 废弃。接收端识别门槛。
