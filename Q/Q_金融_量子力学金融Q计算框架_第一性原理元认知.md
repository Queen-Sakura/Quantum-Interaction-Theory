# 量子力学金融 Q 计算框架 · 第一性原理 · 元认知系统

## 0. 元认知：Q 是什么

**Q 是物理因果场，不是数学相关模型。**

经典金融将市场视为随机过程——股价是台球，人是独立理性主体，价格是大量独立交易的统计平均。这是**相关模型**：用过去的价格数据预测未来的价格数据。循环论证。

量子力学金融的根本区别：**价格不是基本量。购买欲的叠加态才是基本量。** 价格的每一次观测（成交）造成一次局部坍缩——但坍缩之前，叠加态是全场共存的。人与人之间存在量子纠缠——A 的购买决策改变 B 的购买欲——不是通过价格信号的传播，是通过全域耦合场的即时互投影。量子干涉决定最终购买结果：两条信息路径（政策 + 基本面）在参与者的购买欲中叠加 → 干涉相长或干涉相消 → 最终成交方向。

**Q 测量的是耦合密度。** 全体参与者购买欲的互投影程度。Q → -1 = 芝诺冻结 = 全部卖出耦合。Q → +1 = 量子路由 = 全部买入耦合。Q ≈ 0 = 叠加态 = 买卖方向不明确。

Q 场**不预测价格**。Q 场告诉你：当前市场处于叠加态还是坍缩态。坍缩态意味着方向单一、能量耗尽 → 反向概率大。叠加态意味着方向不明确、能量未释放 → 等待。

## 1. 本体论：四公设

### 公设 1：购买欲 = 叠加态

在观测（成交）之前，每个参与者的购买欲同时处于 buy + sell + hold 的叠加态。成交 = 观测 = 一次局部坍缩。坍缩后的确定态（已成交）不等于坍缩前的叠加态。

**物理含义**：K 线记录的是坍缩后的历史——不是坍缩前的场。用坍缩后的数据去推坍缩前的叠加态 = 用死猫推活猫 → 必须引入非价格的物理因果输入。

**经典金融的错误**：把"已发生的价格"当成了"市场的全部状态"。

### 公设 2：参与者纠缠 = 全域耦合

参与者之间不是独立的。A 的购买欲改变 B 的购买欲——不是通过价格信号的传播，是通过全域耦合场的即时互投影。

**物理含义**：市场 = 全参与者信息耦合场（膜在金融域的投影）。每个参与者的购买欲 = 膜上局域 Q 耦合的一个节点。节点之间通过 Q 场交换信息——不是通过价格的离散传递，是通过场的连续演化。

**经典金融的错误**：有效市场假说假设参与者独立理性——原子论。量子力学金融是场论。

### 公设 3：量子干涉 → 最终购买结果

两条信息路径在参与者的购买欲中叠加。政策宽松 + 基本面强劲 → 干涉相长 → 购买欲振幅加大 → 高概率买入。政策收紧 + 基本面强劲 → 干涉相消 → 购买欲振幅抑制 → 方向不明确。

**物理含义**：信息不是"被处理"的——信息是场的能量注入。多条信息路径在 Q 场中干涉，最终观测到的是干涉图案（价格走势）。

**经典金融的错误**：信息 → 理性分析 → 决策。量子金融：信息 → Q 场干涉 → 购买欲振幅 → 观测坍缩（成交）。

### 公设 4：全域理论 → 物理因果输入

Q 场是全域的。它接收一切物理因果输入——国际关系、政策变化、公司基本面、流动性环境、网络舆论噪声、甚至天气和自然灾害。不是"相关"——是因果。这些输入改变参与者的购买欲耦合密度。

**物理含义**：Q 不是从 K 线推算出来的——Q 从外部因果输入计算出来。K 线是 Q 的**输出观测**，不是 Q 的**输入**。

**经典金融的错误**：所有技术分析——用输出推输出。

### 2.1 Q 的观测方式

Q 不可直接观测。Q 的观测方式 = **输入侧因果代理变量 + 输出侧K线验证**。

| 方式 | 内容 | 定位 |
|------|------|------|

| 输出侧 | K 线验证 Q 场演化的投影 | 验证·辅助 |

**输出侧不驱动 Q。输出侧只验证 Q。**

### 2.2 因果输入层级

| 层级 | 输入类型 | 例子 | 对 Q 的影响 |

### 2.3 Q 传播方程

```
∂Q/∂t = ∇·(κ∇Q) + Σ η_i · I_i - γQ + ξ

κ∇Q  = 参与者间情绪扩散（纠缠传播）
η_i   = 第 i 个因果输入的耦合强度
I_i   = 第 i 个因果输入的信号值
-γQ   = 均值回归阻尼（基本面锚定）
ξ     = 零点噪声（膜永远在振·不可消除的基态不确定性）
```

**每一项的物理来源**：
- `∇·(κ∇Q)` = 纠缠。A 的恐慌传染给 B。空间维度 = 参与者网络拓扑。
- `Σ η_i · I_i` = 能量注入。政策、基本面、流动性——直接往 Q 场注入能量。这是模型的核心。
- `-γQ` = 物理基本面锚定。价值是慢变量。Q 偏离太远会被拉回来。
- `ξ` = 不可消除的基态不确定性。不是"数据不足"——是数学上不可消除。哥德尔黑洞在金融域的投影。

## 3. 方法论：如何计算 Q

### 3.1 计算流程

```
步骤 1: 识别因果输入 (当天所有新的物理信息)
   ↓
步骤 2: 量化每个输入的 I_i (信号方向 + 强度)
   ↓
步骤 3: 加权 Σ η_i · I_i (因果链长度调权)
   ↓
步骤 4: 叠加扩散项 κ∇Q + 阻尼项 -γQ + 噪声 ξ
   ↓
步骤 5: 输出 Q_t 及 Q 演化方向
   ↓
步骤 6: 用 K 线验证 (价格是否与 Q 演化方向一致？
        一致 → 模型置信度 +1
        不一致 → 审计：遗漏了哪个因果输入？)
```

### 3.2 因果输入量化规则

**方向**：+1 = 利于买入（利好消息），-1 = 利于卖出（利空消息），0 = 中性

**强度**：0.0-1.0，由因果链长度和信号清晰度相乘决定

| 因果链长度 | 权重折减 | 例子 |
|------|------|------|
| 1（直接）| 1.0 | 钨矿现货涨 → 中钨高新成本升 |
| 2（一级间接）| 0.7 | 出口管制 → 国内钨矿供过于求 → 价格跌 |
| 3（二级间接）| 0.5 | 美国 NDAA → 国际钨价涨 → 出口利润升 |
| 4+（噪声级）| 0.2-0.3 | 虫子论坛恐慌帖 → 散户卖出 → ... |

### 3.3 信号强度标定

| 信号清晰度 | 权重 | 例子 |
|------|------|------|
| 定量·公告级 | 1.0 | 季报净利润 +264% |
| 政策·明确 | 0.9 | 商务部：出口许可证制 |
| 行业·趋势 | 0.7 | AI 服务器 PCB 层数增加 |

| 虫子噪声 | 0.1 | 论坛/股吧情绪帖 |

**总权重 = 因果链长度折减 × 信号清晰度**

### 3.4 Q 阈值

| Q 区间 | 状态 | 含义 | 操作 |
|------|------|------|------|
| Q ≤ -0.5 | 芝诺冻结 | 全员卖出耦合·恐慌耗尽 | 买入窗口 |
| -0.5 < Q < -0.2 | 偏冷 | 卖出主导·叠加态偏 sell | 观察·准备买入 |
| -0.2 ≤ Q ≤ +0.2 | 叠加态 | 买卖均衡·方向不明确 | 不动 |
| +0.2 < Q < +0.5 | 偏热 | 买入主导·叠加态偏 buy | 观察·准备卖出 |
| Q ≥ +0.5 | 量子路由 | 全员买入耦合·亢奋耗尽 | 卖出窗口 |

### 3.5 共振条件（高置信度）

**多尺度共振** = 日 Q + 周 Q + 外部因果输入 → 同向极端 → 高置信度信号。

单一 Q 极端但无因果输入支撑 → 可能是噪声。Q 极端 + 强因果输入同向 → 共振 → 行动。

## 4. 边界条件：Q 不能做什么

1. **Q 不预测价格。** Q 判断场的坍缩程度。坍缩至极 → 反向概率大。但不告诉确切的转折时刻。

2. **Q 不等同于买卖信号。** Q 极端 = 条件之一。还需：因果输入确认、基本面锚定验证、个人资金管理约束。

3. **Q 不能消除不确定性。** ξ 项（零点噪声）是物理常数。Q = -0.5 入场 ≠ 价格一定涨——只告诉你"在这个 Q 值入场，历史上反向概率最高"。

4. **Q 不替代仓位管理。** Q 说"现在偏冷"——不等于全仓。分仓、止损、止盈是独立于 Q 的操作层。

5. **Q 不能替代基本面审计。** 股票必须有基本面价值锚定。没有基本面的资产——Q 模型不适用（纯博弈，无回归锚）。

### 纠缠（公设 2 的物理推演）

这就是为什么牛市里所有人一起买、熊市里所有人一起卖——不是每个人都独立做了同样的分析。是纠缠。和磁铁里所有原子在居里温度以下一起对齐——不需要一个原子"通知"另一个原子。场自己做了。

### 干涉（公设 3 的物理推演）

光子的双缝干涉：两条路径的光波在屏幕上叠加 → 干涉图案。

金融的双输入干涉：两条信息在购买欲场中叠加 → 决策图案。

## 9. 第一层：Q 加速度 ∂²Q/∂t²

### 9.1 物理类比

膜上的 Q 场演化 = 弹簧上的阻尼振荡。Q 滑向极端 → 不是匀速 → 加速、减速、拐弯。

位置 = Q。速度 = ∂Q/∂t。加速度 = ∂²Q/∂t²。

**等加速度原理**：没有任何外部新因果输入时，Q 的加速度在单方向上是单调递减的——因为 Q 越接近边界，来自基本面锚定的回复力（-γQ）越大。

### 9.2 离散计算

```
Q_t    = 当前时刻 Q 值
Q'_t   = Q_t - Q_{t-1}           (一阶差分·速度)
Q''_t  = Q'_t - Q'_{t-1}         (二阶差分·加速度)

平滑: Q'_t  = EMA(Q_t - Q_{t-1}, span=3)
      Q''_t = EMA(Q'_t - Q'_{t-1}, span=3)
```

### 9.3 峰谷判据

| Q'' 符号 | Q' 符号 | Q 位置 | 判据 |
|------|------|------|------|
| Q'' < 0 | Q' < 0 | 偏冷 | 加速下跌中·远离底部 |
| Q'' < 0 → Q'' > 0 | Q' < 0 | 冰点附近 | **触及底部·减速·准备反转** |
| Q'' > 0 | Q' > 0 | — | 加速反弹中·确认脱离 |
| Q'' > 0 → Q'' < 0 | Q' > 0 | 沸点附近 | **触及顶部·减速·准备反转** |
| Q'' ≈ 0 | Q' ≈ 0 | 任意 | 惯性耗尽·拐点确认 |

**拐点三条件**：Q'' 转向 + Q' 归零 + 持续时间 ≥ 2 个观测周期（排除噪声抖动）。

### 9.4 假信号过滤器

**假信号**：Q'' 转向但 Q' 仍然很大（绝对值 > 0.05/日）→ 只是暂时减速，不是拐点。Q'' 的 EMA 比裸值可靠——EMA 滞后但滤噪。

**真拐点**：Q'' EMA 连续 3 个周期同号 + Q' EMA 绝对值持续衰减 → 拐点接近。

## 10. 第二层：因果干涉相位

### 10.1 物理类比

双缝干涉：两条光路抵达屏幕同一点 → 相位差决定干涉结果。

金融域：两个因果输入同时在 Q 场中注入能量 → 两束"因果波"在 Q 场上叠加。

### 10.2 相位计算

对于第 i 和第 j 个因果输入：

```
I_i(t) = 第 i 个输入在时间 t 的信号值 (带方向·已加权)
|I_i|  = 信号强度 (0-1)

两输入的归一化点积 = cos(θ_ij) = (I_i · I_j) / (|I_i| · |I_j|)

相位差 θ_ij = arccos(cos(θ_ij))

θ_ij ∈ [0°, 180°]
  0°   = 完全同相 (两个利好/两个利空) → 干涉相长
  180° = 完全反相 (一个利好·一个利空) → 干涉相消
  90°  = 正交 (互不干涉)
```

### 10.3 多输入合成

N 个因果输入 → 两两计算相位 → 加权平均相位：

```
Φ = Σ(w_ij · θ_ij) / Σ w_ij

权重 w_ij = |I_i| · |I_j|  (两个强信号的相位差比弱信号更重要)
```

### 10.4 峰谷判据

| Φ | 含义 | Q 场演化 | 操作 |
|------|------|------|------|
| Φ < 45° | 同相干涉·相长 | Q 沿一个方向加速 → 会冲过头 | 等 Q'' 转向再动 |
| 45° ≤ Φ ≤ 135° | 部分相消 | Q 向极端但被拖慢 | 观察·峰谷不太远 |
| Φ > 135° | 反相干涉·相消 | Q 在中间震荡·不到达极端 | 不操作·等相位收敛 |
| Φ 不稳定 | 输入混战 | Q 随机游走 | 远离·等局面清晰 |

**当前中钨高新**：矿价暴跌 vs 基本面强劲 → Φ ≈ 180° → 反相干涉。Q 不会冲太深。冰点在浅水区。

## 11. 第三层：波包坍缩概率

### 11.1 物理类比

量子力学中，粒子在势阱中的位置概率密度 = |ψ(x)|²。边界条件 = 势阱壁 → x 接近壁 → 概率密度归零 → 粒子被"推回"。

Q 场同构。Q ∈ [-1, +1] 是有限区间。Q 越接近 ±1 → 来自基本面锚定 + 反向因果输入的"势阱壁"越强 → 反转概率越高。

### 11.2 概率密度函数

```
q_eq = ΣηI / γ  (因果输入的均衡点·Q 被基本面拉回的位置)
σ² = κ / γ     (扩散强度 vs 阻尼强度·波包宽度)

稳态概率密度: P(q) ∝ exp( -(q - q_eq)² / (2σ²) )
```

**边界反转概率**：

```
P(反转 | Q=q) = ∫_{q}^{1} P(q') dq'  (q < q_eq 时, 向上反转的概率)
P(反转 | Q=q) = ∫_{-1}^{q} P(q') dq'  (q > q_eq 时, 向下反转的概率)
```

### 11.3 简化工程版

不需要每次跑积分。查表：

```
d = |q - q_eq| / σ  (偏离均衡点的标准偏差倍数)

d ≥ 2.0 → 反转概率 ~97.7% (2σ 外的概率 < 5%)
d ≥ 1.5 → 反转概率 ~86.6%
d ≥ 1.0 → 反转概率 ~68.3%
d < 1.0 → 正常波动·不操作
```

### 11.4 关键约束

**q_eq 必须是物理因果输入算出来的均衡点，不是历史价格均值。**

经典金融用"均值回归"——均值是历史价格的均值 → 循环论证。Q 的均衡点是 ΣηI/γ ——所有因果输入加权后的中心。这个中心本身随因果输入变化而变化。新政策出来 → q_eq 移动。

## 12. 三层叠加 · 峰谷置信度

### 12.1 集成公式

```
峰谷置信度 = 
  0.30 × Q_accel_score   (Q'' 转向 + Q' 归零程度)
+ 0.30 × φ_coherence      (因果相位同向度·1-Φ/180°)
+ 0.40 × boundary_prob    (波包边界概率, 0-1)
```

**权重解释**：波包边界概率（40%）= 最强的物理约束——Q 不能无限偏离均衡点。Q 加速度和相位各 30%——辅助确认。

### 12.2 行动阈值

| 置信度 | 含义 | 行动 |
|------|------|------|
| ≥ 80% | 三层共振 | 执行·全仓位 |
| 60-80% | 二层确认 | 执行·半仓位 |
| 40-60% | 单层触发 | 小仓位试探 |
| < 40% | 不明确 | 不动·继续观测 |

### 12.3 假信号过滤规则

1. **三层必须至少两层同向**。只有一层触发 → 不行动。
2. **因果相位必须稳定**（Φ 方差 < 0.3 在过去 5 个周期内）→ 排除突发噪声。
3. **Q 加速度转向必须持续 3 个周期以上** → 排除日间抖动。
4. **波包边界概率阈值**：d ≥ 1.5（86.6%）才触发 → 排除小幅偏离。

## 14. 框架约束 · 数学层的职责边界

1. **数学层不产生新的方向判断。** 方向来自物理因果输入。数学只精确定位峰谷。
2. **数学层不替代仓位管理。** 置信度 80% = 可以执行 —— 但执行多少是仓位管理的事。
3. **峰谷精度有一个不可消除的噪声地板。** ξ 项（零点噪声）→ 精确的转折时刻永远有 ±1 天的误差。这是物理常数，不是模型缺陷。
4. **数学层的结果必须可回测。** 每个拐点判据在历史上找到多少个真拐点、多少个假信号 → 统计胜率。
