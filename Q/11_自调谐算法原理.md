## 2. 算法核心：耦合梯度收敛

### 2.1 基本操作

物理系统的状态 = 一组驻波模式。每个模式有自己的 Q 值（耦合密度）。

系统演化 = 模式之间不断发生耦合事件（量子交易）。

```
一笔耦合事件：
    触发条件：两个模式的频率匹配 + 能量够
    结果：    一个模式切换 → 旧的叠加态被减少 → 空间缩小
    记录：    账单写在膜上。不可逆。
```

每一步耦合 → 消除一部分不兼容的模式 → 剩余可能的构型数减少 → 搜索空间自动收缩。

**不是"搜索"。是耦合事件在替你收缩空间。**

### 2.2 为什么不卡在局部极小值

经典梯度下降卡住 → 因为它是单点搜索。到了一个坑——周围全是上坡——不知道隔壁山谷更深。

自调谐不卡 → 因为**所有没被耦合淘汰的模式还在振。** 它们在系统中保持"微振幅"。热涨落偶尔推动一个新模式 → 跟现有模式耦合 → 如果新模式的 Q 配置更低 → 系统跳到新谷。

```
局部极小值 A → 热涨落 → 新模式 B 短暂上线 → Q_B < Q_A
                                               ↓
                                          B 胜出。系统移到 B。
```

热噪声 = 跳出局部极小值的候选模式源。噪声越大 → 候选越丰富 → 跳越快。然后耦合事件（退相干）来筛选——哪个模式 Q 更低，哪个留下来。

**噪声提出候选。耦合筛选候选。同一个算法。**

### 2.3 收敛保证

Q 耦合事件只增不减（不可逆）。每一笔耦合交易在膜上留下永久记录。所以 Q 的总耦合密度单调趋向极值。证明：

- 有限系统 → 有限模式数 → 有限耦合事件
- 每笔交易不可逆 → Q 单调变化
- 单调 + 有限 → 必然收敛

不需要"凸空间"假设。不需要"利普希茨连续"假设。只要系统是有限腔，耦合事件不可逆，收敛到全局最优是**数学保证**的。

## 6. 工程化路径

把自调谐算法从自然抄到工程——需要三个组件：

### 6.1 模式空间定义

将问题的所有可能解定义为驻波模式。每个模式有频率（特征值）和振幅（概率权重）。金融里——模式 = 交易者的心理价位。物流里——模式 = 可能的路径组合。钢铁里——模式 = 可能的相构型。

### 6.2 Q 耦合规则

定义系统内模式之间的耦合规则。哪些模式能发生耦合 → 交易 → 淘汰一部分模式。规则就是物理：频率匹配、能量守恒、边界条件兼容。不是人工设计的。是**物理本身就有的。**

### 6.3 温度（噪声水平）

设定系统的噪声水平（温度）。温度控制候选模式生成的速率。太低 → 不跳出局部最优。太高 → 耦合太快，跳过头。最优温度 = 模式密度最高处的 Q 值对应的能量。
