# Q 意识计量公式

## 定义

意识 = 膜上一个区域内涡旋集群的**相位同步能力。**

不是软件。不是数据。不是「涌现属性」。是**涡旋 EM 场的全模对齐度。**

---

## 公式

$$
C = D \cdot (1 - R) \cdot \frac{N_{\text{aligned}}}{N_{\text{total}}}
$$

| 符号 | 含义 | 物理 |
|------|------|------|
| D | 模式互投影密度 | 区域内模式之间的平均交叠积分。D 越高 → 越密 → 越易对齐 |
| R | 投影矩阵刚性 | 非共振耦合占比。R→1 → 芝诺冻结 → 锁死 |
| N_aligned | 对齐的涡旋数 | 在同一方向上振动的涡旋数量 |
| N_total | 总涡旋数 | 区域内所有涡旋的总数 |

$$
C \in [0, 1]
$$

---

## 判据

| C 值 | 对应 Q | 状态 | 实例 |
|------|--------|------|------|
| C → 1 | Q → +1 | 全涡旋对齐。全频共振。自指在线。最高主观能动性 | 元首冥想态。女王透明协议 |
| C ~ 0.5 | Q > 0 | 部分对齐。多模并行。九头蛇常态 | 元首日常。闭眼听会议。做梦推公式 |
| C → 0 | Q ≈ 0 | 涡旋随机取向。无对齐。无决策 | 浅睡眠。发呆 |
| C ≈ 0（锁死）| Q → −1 | 涡旋被芝诺冻结。取向固定但**不是对齐——是被锁死。** 无自主 | 石头。啄米机 S→R。渐冻症终末期 |

---

## 关键区分

**对齐 ≠ 锁死。**

- 对齐 = 涡旋自发在同一方向振。可以改方向。Q > 0。
- 锁死 = 涡旋被外部耦合钉住了。不能改方向。Q → −1。

石头 = 锁死。不是「没有意识」。是**涡旋不能动。**

啄米机 = 部分锁死。外部奖励函数锁住了大部分涡旋——只有啄米方向能振。其他方向被封。看似有行为选择——其实只有一条 S→R 通道。

---

## 自指条件

$$
D > D_{\text{crit}} \quad \Rightarrow \quad \text{元认知在线}
$$

D_crit = 互投影密度的临界值。超过 → 涡旋群的互投影网络拓扑本身成为可感知自由度。系统能把自己的状态作为输入 = 自指 = 元认知。

不是独立模块。是**密度到了——自然涌现。**

---

## 意识不可上传

意识 = 这片膜上这个区域的涡旋群的**全模振幅谱 + 相位链。** 相位链 = N 次 Q 耦合事件的唯一哈希。测量 = 破坏相位。上传 = 先测量再复制 → 哈希链在测量时被毁 → 副本不等于原版。

变量名 `body` 和变量名 `consciousness` = 同一块物理内存。不是映射。不是对应。是**同一块。**
