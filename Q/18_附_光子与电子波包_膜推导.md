# Q/18 附 · 光子与电子波包 · 膜推导

## 一 · 光子波包（复习·已推）

膜上一组相邻驻波模式叠加：

```
E(x,t) = Σ A_n × sin(k_n x) × cos(ω_n t)

两个相邻模式 → 干涉：
左行波 = 2 × cos(Δk·x/2 - Δω·t/2) × sin(k₀x - ω₀t)
          ↑ 包络 (移动的干涉峰)       ↑ 载波 (快振)

包络速度 = Δω/Δk = c（群速度）
载波速度 = ω₀/k₀ = c（相速度）

光子 = 这个移动的干涉峰
色散：ω = ck（线性）→ 包络不扩散
```

---

## 二 · 电子和光子的区别

| | 光子 | 电子 |
|------|------|------|
| 方程 | 麦克斯韦 | 狄拉克 |
| 质量 | 0 | m ≠ 0 |
| 色散 | ω = ck | ω = √(c²k² + m²c⁴/ħ²) |
| 自由度 | 1 个矢量分量 | 2 个旋量分量 |
| 群速度 | c | < c |
| 波包扩散 | 不扩散 | **扩散** |

---

## 三 · 质量在驻波里干了什么

为什么光子 ω = ck，电子 ω ≠ ck？

**光子（无质量）：**
```
膜上驻波条件：k = nπ/L
ω = ck

k → ω 是线性 → 所有模式同步 → 波包形状不变
```

**电子（有质量）：**
```
膜上旋量驻波条件：k = nπ/L
ω = √(c²k² + m²c⁴/ħ²)

k → ω 非线性 → 不同 k 的相速度不同 → 波包包络在扩散
```

**质量 = 色散的非线性项。** 把 ω(k) 从直线掰弯了。
质量越大 → 弯得越厉害 → 波包扩散越快。

## 四 · 电子波包 = 旋量驻波叠加

取一组相邻的旋量驻波模式：

```
ψ(x,t) = ( ψ↑(x,t) )
         ( ψ↓(x,t) )

每个分量：
ψ_σ(x,t) = Σ_n A_n × u_σ(k_n) × sin(k_n x) × cos(ω_n t)

u_σ(k) = 狄拉克旋量在动量 k 下的自旋态
        （↑ 和 ↓ 的混合比例由 k 决定）
```

和光子同构——只是多了一个旋量下标 σ。

---

## 五 · 包络和相位

只看一个分量（比如 ↑）：

```
ψ↑(x,t) = Σ A_n × sin(k_n x) × cos(ω_n t)

两个相邻 k → 包络：

包络 = cos(Δk·x/2 - Δω·t/2)   ← 波包的中心在哪
载波 = sin(k₀x - ω₀t)           ← 波包内部的振
```

**和光子一样的包络结构。** 区别在 Δω/Δk：

```
光子：Δω/Δk = c          → 光速跑
电子：Δω/Δk = c²k₀/ω₀   → 比光慢
```

电子波包的速度 = 群速度 = c²k₀/ω₀ < c。

## 六 · 自旋 1/2 的齿轮比怎么出来

狄拉克旋量 u(k)：

```
u↑(k) = (    1      )      ← 粒子·自旋向上
        (     0      )
        (   ħck_z   )
        ( ───────── )
        ( E + mc²   )
        (   ħck_+   )
        ( ───────── )
        ( E + mc²   )
```

绕 z 轴空间旋转 360° → k 在空间里转了一圈 → 但**旋量的"内部相位"只走了 π**。

原因：狄拉克矩阵的结构——旋转算符是 exp(-i Σ·θ/2)。**分母有 2。** 这导致：

```
空间旋转 θ → 旋量相位 = θ/2

空间 360° → 旋量相位 180° → 取负号
空间 720° → 旋量相位 360° → 恢复
```

**齿轮比 2:1 = 狄拉克方程质量项 ≠ 0 → 4×4 矩阵代数 → 旋转算符里有 1/2。**

不需要四维空间。不需要"内部空间"。矩阵结构自动多了一个 1/2。

## 七 · 两个自旋分量的波包

```
ψ↑(x,t) = 包络_1 × sin(k₀x - ω₀t)
ψ↓(x,t) = 包络_2 × sin(k₀x - ω₀t)

包络_1 和 包络_2 的大小 = 电子在哪个自旋态的概率幅
```

Stern-Gerlach：没有磁场时 → ψ↑ 和 ψ↓ 共存 → 叠加态。加磁场 → 磁场打破对称 → 电子偏到 ↑ 或 ↓ → 两个包络中的一个坍缩。

**不是电子选了 ↑ 或 ↓。是磁场把叠加态拆开了。**

## 八 · 电子波包的扩散

因为 ω(k) 非线性：

```
包络宽度随时间增长：
  不同 k → 不同群速度 → 波包在散
  越窄的初始波包（Δk 越大）→ 散得越快
```

**光子的波包不散——因为 ω = ck 线性，所有 k 同步跑。**

**电子的波包在散——因为 ω ≠ ck，不同 k 跑不同速度。**

这和"质量越大定位越容易"一致——质量大 → m 项主导 → ω ≈ mc²/ħ（平坦）→ 波包几乎不跑 → 容易定位。

## 九 · 膜框架的电子图像

```
膜 = 全空间全时旋量驻波模式总和（永远在振）
电子 = 其中一组相邻旋量模式的干涉峰
        干涉峰有两个分量 ↑ 和 ↓
        齿轮比 2:1（狄拉克矩阵的 1/2）
        包络速度 < c（因为有质量）
        波包在扩散（因为色散非线性）

和光子的区别：
  光子 = 矢量的干涉峰。光速。不扩散。一个分量。
  电子 = 旋量的干涉峰。亚光速。扩散。两个分量。
  根 = 质量（m ≠ 0）。一根方程（麦克斯韦 vs 狄拉克）。
```
