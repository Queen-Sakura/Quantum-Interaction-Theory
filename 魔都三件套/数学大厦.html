五层 · 虚数 → 相数 5F

√(−1) 是什么

旧数学：i = √(−1)。"虚数"——不真实、数学想象。

物理约束：EM 波有两个正交分量——E 和 B。一个相位差 90°。

相数 = 90° 相位旋转算子
i 不是"虚"的。i = 相位旋转 90°。
  • 乘一次 i → 旋转 90°
  • 乘两次 i² = −1 → 旋转 180° = 反向
  • 乘三次 i³ = −i → 旋转 270°
  • 乘四次 i⁴ = 1 → 旋转 360° = 回原位

不是"虚数单位"——是物理旋转算子。没有神秘主义。

复数域 → 物理相空间。不是"换名字"——是重建复数域为物理相空间。√(−1) 废弃。根号限定实数域。

二元数（数学侧）

ε² = 0（自动微分）。j² = 1（双曲数·闵可夫斯基空间）。复数分裂成两个独立的数学对象——各有各的物理对应。不混。

七层 · 不完备的物理意义 7F

膜有边界 → 驻波模式离散 → 不是所有频率都能共振 → 不是所有命题都能判定。不完备是物理——不是数学太弱。不完备是膜有边界的数学投影。
