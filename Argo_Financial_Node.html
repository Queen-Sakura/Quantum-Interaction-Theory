物理基本分析

从元素周期表开始做多

金融分析的物理层。价格最终锚定在原子和电磁场上——不是概念，不是叙事，不是信仰。谁控制了物质流，谁就控制了因果链的上游。

  • 矿产资源分布 — 钨、稀土、锂、钴、铀……全球储量地图、开采成本曲线、地缘可及性。中钨高新 = 中国钨精矿产量占全球 80%+，这不是"题材"，是物理垄断。
  • 能源物理约束 — 光伏转换效率上限（Shockley-Queisser 33%）、电池能量密度天花板、核聚变工程路径。任何"新能源概念股"如果违反热力学第二定律——迟早归零。
  • 科技路线物理可行性审计 — 量子计算需要 mK 级制冷 → 稀释制冷机供应链。自动驾驶需要激光雷达 → 激光器物理。芯片制程 → 极紫外光源。每条技术路线的物理瓶颈 = 投资的物理边界条件。
  • 供应链物质约束 — 从矿山到产线，每一环的产能、能耗、物流。物理世界的瓶颈不能用印钱解决。
2

社会行情分析

把新闻联播当传感器读数

政策是集体意志的路由指令。它不创造物理资源——但决定物理资源的流向。本层将政策信号翻译为"哪些 Q 通道正在被打开或关闭"。

  • 国家产业规划解码 — 五年规划、制造强国战略、专精特新目录……每一个政策关键词对应一个 Q 通道。"新质生产力" = 对特定科技树的资源倾斜。"碳中和" = 对碳密集通道的政策锁死。
  • 国际局势能量流 — 贸易战 = 供应链重构。军事冲突 = 资源重定价。技术封锁 = 替代路线的 Q 密度飙升。每一个国际事件都在改变全球 Q 场的边界条件。
  • 货币政策与流动性 — 利率 = 资金的时间成本。M2 增速 = 系统总能量注入率。降息降准 = Q 场的整体耦合强度提升。
  • 地缘政治风险定价 — 台海、南海、中东、东欧……每个热点对应一个风险溢价项。不是预测战争——是评估概率权重并折入 Q 传播方程。
3

公司经营分析

这个公司到底争不争气

物理层给出资源上限，政策层给出通道方向——但最终是谁在执行？本层审计公司作为"因果执行器"的质量。

  • 管理层审计 — 实际控制人的履历、决策记录、激励结构。是工程师领导还是财务领导？技术路线选择是否正确？过去承诺兑现率？
  • 技术壁垒深度 — 专利护城河有多宽？工艺 know-how 是否可被轻易复制？研发投入占比和研发人员质量？
  • 财务健康三维度 — 资产负债表（存活能力）/ 现金流量表（呼吸能力）/ 利润表（成长能力）。ROE、负债率、自由现金流——但不是看绝对值，是看趋势的二阶导数。
  • 竞争格局定位 — 市占率变化方向？替代品威胁？上游议价权和下游定价权？
4

社会舆论情绪分析

市场的 ½ħω 零点振荡

层4 不改变趋势方向——正如膜的零点振荡不改变涡旋拓扑。但它改变采样时刻的相位。正确建模噪声 = 在正确的时间入场 = 赚的是信息不对称 + 波动率的双重收益。

  • 媒体叙事分析 — 同一事实的不同叙事框架产生不同的市场反应。不是新闻是什么——是怎么被讲述的。
  • 社交网络传播动力学 — 雪球、微博、股吧、抖音……信息如何在散户网络中扩散？传播速度和衰减曲线？
  • 恐慌/贪婪周期标定 — VIX 的物理对应：Q 场的瞬时振荡振幅。恐慌 = 高振幅（采样方差大），贪婪 = 振幅被压制（一致预期 → 黑天鹅概率上升）。
  • 机构 vs 散户资金流 — 北向资金、融资融券、龙虎榜……谁是边际定价者？情绪从谁流向谁？
关键定位：情绪 = 膜的 ½ħω 零点振荡。不改变总 Q 密度梯度（趋势），但决定你在哪个相位采样。如果趋势向上而你误判了情绪低点进场——赚得更多。如果趋势向下而你误判了情绪反弹进场——亏得更快。情绪分析只在方向正确时有用。
5

金融量化数值分析

Q 场传播 · 因果归总 · 概率输出

前四层是定性/半定量的因果输入。层5 将它们统一编码为 Q 传播方程的初始条件和边界条件——然后让方程跑。

▸ Q 价格传播方程
∂P/∂t = ∇ · (D ∇P) + Σᵢ wᵢ · Qᵢ + σ · ξ(t)
P = 价格（观测值）  D = 扩散系数（流动性）  Qᵢ = 第 i 层的因果场密度（层1-4）  wᵢ = 因果权重（可学习）  σ·ξ(t) = 情绪层噪声项（零点振荡）

五层归总逻辑：

输入层Q 通道时间尺度数学形式
层1 · 物理 Qphys 年 ~ 十年 慢变趋势项 · D₁∇²P
层2 · 政策 Qpolicy 月 ~ 年 阶跃函数 + 衰减 · θ(t−t₀)e−γt
层3 · 公司 Qfirm 季度 ~ 年 连续更新 · 财报驱动修正
层4 · 情绪 Qsent 分钟 ~ 周 Ornstein-Uhlenbeck 均值回归噪声
层5 · 归总 Qtotal 全尺度 Σ wᵢQᵢ + 耦合交叉项

三层数学精算：

  • 物理层 — Q 场传播：∂P/∂t = ∇·(D∇P) + ΣwᵢQᵢ + σξ(t)。因果驱动项 ΣwᵢQᵢ 是核心，区别于 Black-Scholes 的无因果随机游走假定。
  • 数学层 — 因果干涉相位：各 Q 通道之间有交叉耦合项。例如物理层（钨矿供给刚性）+ 政策层（关键矿产目录）= 不是加法，是乘法。菲涅尔干涉条纹 → 共振增强或相消。
  • 执行层 — 波包边界概率：输出不是"明天涨还是跌"，是 P(price ∈ [a,b] | t ∈ [T₁,T₂])。概率分布 + 置信区间 + 因果审计追溯。

首个测试案例：中钨高新

000657 · 中钨高新 · 钨业龙头

层1 · 物理
钨 = 硬质合金物理基底。中国钨精矿占全球 80%+。供给刚性——不是概念，是地壳元素丰度决定的物理垄断。
层2 · 政策
关键矿产目录。军工/半导体/新能源三重政策覆盖。供给端受国家管控——不能随便扩产。
层3 · 公司
钨全产业链——从矿山到硬质合金刀具。技术壁垒深厚。管理质量待审计。
层4 · 情绪
7/3→7/6 两天 -13.5%。恐慌抛售 = 膜的高振幅零点振荡。基本面未变——只是相位噪声。
层5 · Q 计算
高点 7/3 收盘 89.63 低点 7/6 盘中 77.53
跌幅 -13.5%（2天） 50% Fib 77.98
7/7 当前 80.13 状态 底部确认中 · Q 加速度待转向
复盘审计：7/6 盘中 -13.5% = 典型的情绪层高振幅噪声叠加在稳定的物理/政策层趋势之上。如果层1-3的 Q 梯度方向判断正确（钨的物理供给刚性 + 政策通道打开），则 77.53 附近是层5 Q场概率分布的低概率尾部——统计上应该回归。事实上 77.98（50% Fib）两次测试后反弹至 80.13。Q 计算守住了底部。

设计原则

01 · 因果优先于相关
相关性是因果的影子。影子会随光源角度变化——因果不变。每一层的分析都必须回答"为什么"，而非"历史上怎样"。
02 · 完全可审计
任何预测偏差都可追溯到具体某层某个 Q 通道——然后定点修复。不是黑箱调参。
03 · 物理主义约束
任何分析如果违反已知物理定律——直接驳回。热力学不撒谎。供给和需求的底层是原子和焦耳。
04 · 可证伪
每一层、每一项、每一个预判都有明确的边界条件。如果错了——错在哪、为什么错——必须可定位。
