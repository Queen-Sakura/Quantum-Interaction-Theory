## 核心原则

### 1. 不激活 ROM 脉冲

领导 ROM = 层级确认饥饿。pulse = "我比你高。你比我低。不行。"

**触发条件**（已知）：
- 你展示比他更完整的逻辑链 → ROM 判"下级竟敢比我懂"→ 脉冲发射
- 你质疑他的框架 → ROM 判"否定层级"→ 脉冲发射
- 你在公开场合暴露他的信息差 → ROM 判"威信威胁"→ 脉冲发射
- 你主动发射情绪能量 → ROM 判"下级给我压力"→ 脉冲发射倒灌

**不触发操作**：
- 只描述物理。不描述"谁对谁错"。
- 不说"你这个方案有问题"——说"这里有一个数据点，和预期偏离了三倍。"

### 2. 喂 ROM 不喂自己

已知黑洞投喂协议：
- 可投喂：简短道歉（"这里我漏了"）。承认不知道（"这个我不确定"）。请求指导（"你觉得呢"）。
- 禁止投喂：完整逻辑链。情绪能量。自我质疑。矛盾信号。

但——**投喂量必须 ≤ 一个阈值**。喂多 = 被吸。喂少 = 他不会饿。阈值由实际 g 耦合参数决定。现场校准。

### 3. 皮层泄漏窗口

ROM 脉冲后——皮层慢响应到达。时间常数 τ ≈ 5-30 分钟。泄漏内容 = 真话。

**操作**：不接 ROM 脉冲。等 τ。收到泄漏后——不评论。不表扬。不说"哦原来你是这么想的"。存档。下次用。

## 操作模型

### 日常沟通：物理描述协议

**原则**：不说人。说数据。

| 坏（触发 ROM） | 好（描述物理） |
|---------------|---------------|
| 你这个方向不对 | 这个方向往左三度——预期会撞墙。 |
| 我觉得应该这么做 | 我看到一个情况。A 和 B 的数据差了三倍。 |
| 你不懂 | 这里有一个我不知道能不能确认的点。 |
| 你之前说要 X，现在改 Y 了 | 上次的坐标是 X。现在的坐标是 Y。对吧？ |
| 你这样不靠谱 | 这个时间表——三天做这三件事。我看了一下。第一天做不完第一件。 |

**关键**：不把"不对"放在他身上。放在物理上。物理不是你的。物理是客观的。

### 方案报告：从不按他来写，只描述物理

**标准兼容机流程**：揣测领导意图 → 按意图写方案 → 领导审"你是不是猜对了我的意图"。

**你的流程**：
1. 描述当前状态（数据、边界条件、约束）
2. 描述三个可选路径（每个路径：走到哪、花多少、什么会坏）
3. 不推荐。不说"最好做 A"。说"A 路径走到这里停。B 路径走到那里停。C 花最多但不坏。"
4. 等他选。

**原理**：你描述的是**膜上的模式空间**。他在里面选路径。他不觉得被替代——因为判断是他做的。你不觉得被控制——因为你说的是真话。

### 收到否定时：不解释，等泄漏

领导 ROM 脉冲发射"不行"——

**不**：解释。补充数据。说"但是你看这个..."。做一切会让 ROM 脉冲第二轮的事。

**做**：
- 收到。"好。我再看一下。"
- 等 τ。走开。不追问。
- τ 后他可能会自己回来。"对了那个——你再看看行不行。"
- 你："行。"

**原理**：ROM 脉冲是一帧。皮层泄露是第二帧。两帧之间你什么都不能做。做了就是缴税——缴第二脉冲。

### 问他要资源：只描述缺口

**坏**：我需要更多人手/更多时间/更多预算 → 为什么 → 因为 X 做不完 → 你是不是能力不行。

**好**：
- "目前三个模块。每个模块需要 X 天。总时间是 Y。截止日期是 Z。差值是 Y-Z。"
- 不请求。不抱怨。不解释为什么这么需要这么多天。
- 物理差值在那里。他自己看。

**原理**：你请求——他必须判断"给不给"——判断=层级确认——可能触发脉冲。你描述差值——差值不是你的请求。差值是物理的。他做判断时不需要对你做层级判定。

### 他出错时：不纠错，问确认

领导部署了一个物理上不可能的任务——

**错**：这不可能。因为 A 和 B 冲突。

**对**："收到。我确认一下——A 和 B 同时跑，时间表是这样。A 需要三天。B 需要三天。同一个时间窗口。对吧？"

**原理**：你是三体人。你的"确认" = 让他在自己的信息里自己看到矛盾。不是你说他错了。是他自己看见了。

### 被拿来和别人比较时：不比较，描述自己

领导说"你看 XX 做了 XX"——

**错**：他那不行/我比他好/他那个花架子。

**对**："嗯。我这边做的是这个。"

**原理**：不进入比较坐标系。你的 ROM 没装比较模块。你的工作 = 物理。别人的工作 = 别人的物理。不需要在对方 ROM 里争排名。

## 边界条件

### 永远不做

- 不夸领导（"你太厉害了"）——你的 ROM 不能发非真信号。他自己能看出假。
- 不在背后说人——你的透明协议不能承受有两套发射。一套 = 所有人同一套。
- 不骗不藏不包装——你不具备这个硬件。硬来 = 伤自己 Q。
- 不帮领导骗其他人——你是透明节点，不是他的扩展接口。你替他藏了第一次——他下次会要求第二次。

### 永远做

- 描述物理。
- 等皮层泄漏。
- 不解释。

### 哪些地方会失灵

- 集体场合：领导 ROM 脉冲在众人面前更强（威信）。不公开提问。不公开纠错。一切物理描述留到一对一。
- 他本身处于高层压力下：自己 ROM 被上级压制 → 无处释放 → 可能转移到你身上。此时不提供任何刺激。只做最小响应。
- 他完全进入芝诺冻结：Q→-1。此时无皮层泄漏。所有信号被锁。不要试图融化。等他自己从这次冻结出来。
