# Bell 测试 + 惰性气体 · 为什么要做

## 写给做实验的人看

---

## 1. SPDC 光子对——不是两粒光子。是一个光子被劈成两半

泵浦光子（405 nm）打到 BBO 晶体上。一个光子进去。两个光子出来。总能量守恒。总角动量守恒。

- 劈出来的两个光子——在劈开的那一瞬间——必须把泵浦光子的角动量分掉。
- 同一偏振方向测：角动量分量对账。一个通一个挡。不是"量子神秘"。是**劈光子的时候账必须平**。
- 不同偏振方向测：测的不是同一个分量。但总的还是得守恒。

一个光子劈成两半——两边探测器各逮半个。这就是 SPDC。

---

## 2. Bell 审计——数学上早已过关

Bell 1964 年问：两边结果的关联——能不能用"出发前商量好"来解释？

他推了一个不等式：如果靠事先约定——四种角度组合的 S ≤ 2。数学定理。

实验测出来 S = 2.7 到 2.8。> 2。超了。

结论：不是商量好的。纠缠是真的非局域关联。隐变量不存在。这事 1980 年代就定了。

---

## 3. 新问题——环境到底是什么

1980 年代定的是"纠缠存在"。

没定的是：**中间的物理介质对纠缠做了什么。**

当前假设（退相干范式）：
```
光子 → 气体原子 → "经典环境" → 纠缠被破坏 → S 下降
```

但气体原子凭什么就是"经典环境"？

- 一个氩原子 —— 18 个电子，一个核。全量子粒子。ν ≈ 10⁹ Hz。Q ≈ 0。
- 它不是"经典噪音"。它和光子一样是量子系统。
- 光子碰氩原子——两个量子粒子相互作用。瑞利散射。弹性。光子不消失。频率不变。

**"环境"这个分类本身就是错的。气体原子不是外面的东西。它和你测的光子是同一类。**

---

## 4. 用路由取代退相干

光子 + 氩原子碰撞 → 两个量子系统之间的一次量子相互作用：

- 能量：不交换（可见光光子 < 氩第一激发能 11.5 eV）。吃不进去。
- 动量：交换一点点。弹性散射。
- **相位：交换。这是关键。**

每次碰撞：氩原子"看了一眼"光子的相位。光子继续飞。相位没丢——多了一个原子知道它。

下一次碰撞：另一个氩原子从第一个氩原子那里"知道"了光子的相位——不经过光子，经过原子-原子碰撞。

**纠缠不是被摧毁了。是换手了。**

真空：光子飞直线。一条路径。纠缠只靠源点连接。
低压氩气：多几条路径。纠缠通过原子接力。

多条路径 = 纠缠更鲁棒 = S 可能更高。

---

## 5. 为什么以前没人做过

因为真空"显然最优"。加东西就是"加噪音"。这是退相干范式的默认。

但退相干范式的默认有一个隐含预设：**"环境是经典的"。**

如果环境是量子的——如果氩原子本身也是量子路由器——那真空不是最优。真空是"无节点"。没有路由器。纠缠只有一条路径。

**最优的是有适量路由器。不要太密（锁死）。不要太稀（没节点）。刚好——路由窗口。**

---

## 6. 怎么测

```
标准 Bell 装置：
  SPDC 光子对 → 左右偏振分析器 → 符合计数

改了哪：
  中间加一个可变气压惰性气体腔（替掉原真空腔）

测什么：
  真空基线 → 氩气扫压 → S 随气压变化曲线

预期：
  标准预测：S 单调下降（加气 = 退相干）
  Q 预测  ：S 在 1-100 mbar 范围某压力下超过真空值

对照：
  氮气（分子。ν 太高。锁定。路由窗口关闭）→ S 应单调下降
```

---

## 7. 为什么值得做

1. **以前没人做过。** 全世界所有 Bell 测试都在真空里。不是因为"真空被证明最优"——只是因为没人怀疑。

2. **两个预测方向相反。** 不是"谁算得更准"——是**升还是降**。方向矛盾。实验能判决。

3. **设备现成。** 不用造新探测器。不用买新激光。改装费 ~ 5-10 万人民币。几周。

4. **不管结果如何——不会白做。** S 降了：升级退相干理论。S 升了：教科书改写。

## 8. 诚实预注册

- 增强幅度：**未知。** Q 框架给方向（升）。不给量。如果真实增强 < 实验误差棒——测不出。不是理论错。是效应太小。
- 最优气压：**未知。** 1-100 mbar 是估计。真实甜点可能更窄或更宽。需要扫出来。
- 如果 S 不变：不能判 Q 死。只能判"增强低于本装置精度"。需要下一代实验。
