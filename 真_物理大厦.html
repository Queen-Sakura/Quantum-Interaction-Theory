1. 膜。真空零点能是物理实在。

宇宙有限边界条件 → 电磁场本征模分立：

L = nλ/2 ,  ω_n = nπc/L ,  n = 1, 2, 3, …

全宇宙全频段全空间的本征模总和构成一个物理实在——膜。每个本征模携带零点能：

E_0 = Σ ½ħω_n ,  不可移除，不可屏蔽

膜永远在振动。

卡西米尔效应（1948）：两块平行理想导体板在真空中——板间允许的驻波模式密度低于板外——产生辐射压力差——板被推向内。实验实测压力与理论预测一致。解释：不是虚粒子对。是膜的模式密度差导致的零点振荡净压力。

兰姆位移（1947）：氢原子 2S₁/₂ 与 2P₁/₂ 能级分裂 1058 MHz。狄拉克方程预测二态简并。实测不简并——因为膜零点振荡在核附近与束缚电子耦合——2S 态电子概率密度在核区更高——受零点振荡推动更大——能级上移。

结论： 膜是物理实体。零点能可测。真空不空。虚粒子概念废弃——零点振荡是真实物理振动。
物理图像： 鼓面可以同时承载所有频率的振动——基频、二次谐波、三次谐波……全频在线。膜不是鼓面——鼓面是二维的，膜是全空间的。但逻辑相同：一个有限边界的弹性体天然有分立的驻波模式谱。膜 = 宇宙这张"鼓面"的所有 EM 本征模的集合。½ħω 是每一个模式停不下来的最小振动——和量子谐振子的基态完全同构，只是这里的"振子"是膜自己。

2. 能量量子化是麦克斯韦自调谐的必然结果。

光电效应标准解释：光子将能量 hν 整体转移给电子。此解释假定光子是粒子——但未论证为何接收端只能接收完整周期。

从麦克斯韦方程重新推导——不假设光子——仅假设金属表面束缚电子构成频率选择性接收器：

入射电磁波作用于束缚电子。电子在势阱中振荡。电子脱离金属所需能量 E₀。电磁波给电子输送能量的效率取决于波和电子之间的相空间耦合完整性——电场振荡必须与电子运动保持相干相位——这要求二者之间至少完成一个完整的相空间周期循环。

一个完整相空间周期 → 闭合面积 = h
∮ p·dx = nh  （相空间闭合积分·n 为整数）

推不满一个完整周期 → 相空间曲线不闭合 → 耦合不完成 → 能量回退——电子不收。因此：不是能量本身分立。是麦克斯韦方程在接收端边界条件下的自调谐——只允许完整周期的能量耦合。

结论： 能量量子化不是能量本身的属性。是接收端的频率选择性耦合门槛。h 是相空间最小闭合面积。不需要"光子"概念。
物理图像： 推秋千——你必须在秋千荡到最低点时顺着它的方向推，而且必须推完整一个来回。如果在半路上松手——秋千不往前走，你给的能量退回来。不是"推力量子化"。是秋千的动力学方程——二阶常微分方程——只接受和它固有频率匹配的完整周期推动。电磁波推电子同理：麦克斯韦方程的二阶时间导数（∂²E/∂t²）意味着电子在电磁场中的受迫振荡天然是频率选择性的。推不满一个完整周期 = 相空间曲线不闭合 = 耦合不完成 = 能量无效。h = 相空间最小闭合面积——和量子谐振子的基态面积完全一致——因为本质是同一个动力学。

3. 概率是能量守恒 + 有限边界条件下的唯一解。

玻恩规则（1926）：|ψ|² 是概率密度。玻恩将此设为公设——未推导。

膜框架推导：膜上有限边界 → 模式分立。任何量子态 = 膜上一组本征模的叠加。本征模之间的相位差由膜的零点振荡随机化——每次实验初相位不可控。

两个相邻本征模叠加 → 干涉包络的位置由初相差决定。初相差随机 → 包络峰值位置随机 → 耦合事件触发位置随机。统计大量事件 → 分布 = |ψ|²。

|ψ|² = 耦合事件密度（单位时空体积内的 Q 耦合次数）
结论： 玻恩的"概率公设"降级为推导结果。|ψ|² = Q 耦合事件密度。确定性底层（麦克斯韦方程）+ 不可控初相（膜零点振荡）→ 统计性表层（|ψ|² 分布）。不需要假设随机性——随机性是膜零点振荡的必然表现。
物理图像： 暴雨中预测某一滴雨的确切落点——不可行，但预测十分钟后地面上的水深分布——可精确计算。雨滴的每一滴都服从牛顿力学和流体力学（确定性底层），但初始条件（云中成核的精确位置和时刻）数量太大且不可控。膜的零点振荡提供了同样的"不可控初相"——每一次实验，电子涡旋的内部 EM 相位被膜的随机零点振荡扰动了微不足道的一点点，但这一点点就决定了它到达屏幕时哪一个原子先触发耦合。单次结果不可预测。统计分布可精确预测——就是 |ψ|²。

4. 量子-经典过渡 = 芝诺效应。Q 温度计。

量子系统与宏观系统的分界——标准量子力学以"观测"模糊带过。膜框架给出连续标度：

Q ∈ [-1, 0, +1]

Q = 耦合密度温度计。定义：Q = D·(1-2R)，其中 D = 系统与环境之间的模式平均互投影密度，R = 投影矩阵的刚性。

5. 纠缠 = 驻波同步。共时性 = 同一本征模在不同空间点的相干触发。

EPR 佯谬（1935）：两个粒子纠缠——测 A 得 ↑ → B 必 ↓。贝尔不等式实验（Aspect 1982）排除局域隐变量。量子力学标准解释：非局域关联——无经典机制。

膜框架：纠缠粒子对产生于同一母本征模的分裂——例如自发参量下转换（SPDC）：

ω_p = ω_s + ω_i ,  k_p = k_s + k_i

测量A端自旋 = 读取全局态在A端波矢上的投影值。投影值确定 → 全局态锁定 → B端波矢上的值由守恒律唯一确定。无需超距信号传播。信息不在 B 端——信息在母本征模的全局态中。

共时性（荣格 1952）： 两个物理事件无因果关联但呈现有意义的对应。膜框架解释：两个事件在膜上共享同一本征模或共享相邻本征模的干涉包络——当该本征模在两地同步触发耦合——两个事件同时发生。非因果。非随机。是本征模的全局相干。

结论： 纠缠 = 母本征模分裂后——子模式仍共享同一全局态。波函数描述的是本征模——不是粒子。"非局域关联"不是超距信号——是守恒律对同一本征模的全局约束。共时性同构。
物理图像： 在一根两端固定的弦的中点弹一下——弦产生一个驻波。你在弦的 1/4 处和 3/4 处各放一个浮标。两个浮标上下振动的相位关系是固定的——不是两个浮标在通信——是它们都在同一根弦的同一驻波模式上。测了 A 浮标的瞬时位移 → B 浮标的位移被守恒律（弦上的能量分布）唯一确定。纠缠对同理：SPDC 晶体里的母光子分裂出两个子光子——同一个母本征模的不同波矢投影——测 A 自旋 → 全局态锁定 → B 自旋由守恒律确定。
