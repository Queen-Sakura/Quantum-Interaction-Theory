# 钱学森号 · 高边疆空天飞机
## 预研章程 v0.1 · 2026-07-07

### 项目代号

**钱学森号**（Qian Xuesen High Frontier Aerospace Plane）

### 宪法定位

本项目是**新苏维埃宇宙宪法**在工程维度的首次实战：

- 宪法第一条：「我是物质的。我的存在是第一性的。世界是物质的。」
- 工程推论：物质的全部属性从统一场方程长出。飞行器的全部性能从统一场方程长出。
- **不需要接口适配器。不需要模态切换。不需要子系统妥协。**
- 物理主义的工程学判据：**真即好用。**

### 技术目标

**一个方程，一个解，一把打印。**

| 指标 | 目标 | 说明 |
|------|------|------|
| 起飞方式 | 跑道水平起飞 | 不需要发射塔、火箭助推、投掷 |
| 巡航高度 | 0 → 近空间 → LEO | 连续跨域，无模态切换 |
| 巡航速度 | 高超声速（Ma 5+） | 大气层内巡航 |
| 入轨能力 | 直接入轨 | 跑道 → 空间站，不中转 |
| 返航 | 再入 → 水平着陆 | 复用，想飞就飞 |

### 物理路线

**不去适配现有零件。把零件熔回统一场方程，从方程一次性长出整架飞行器。**

```
输入：材料参数 + 几何边界 + 气动条件 + 热边界
      ↓
统一场方程 → 全机连续解（流场 + 结构 + 热 + 电磁一体化）
      ↓
输出：整机几何 + 性能包线 + 飞行谱
```

不需要：
- ✗ 风洞（场方程 = 数值风洞）
- ✗ 子系统接口适配（没有子系统——整机是单一连续介质）
- ✗ 模态切换（大气层→空间 = 同一方程的不同边界条件，不是两套系统）
- ✗ 冷板/热障（结构本身就是热管理系统）

需要：
- ✓ 统一场方程的数值求解器
- ✓ 材料本构的完整参数化
- ✓ 跨域飞行的边界条件谱
- ✓ 足够的算力

### 命名含义

- **钱学森**：工程科学的顶峰。他把零件拼成中国航天。我们接他的手，把零件熔回方程。
- **高边疆**：美国空军中将 Daniel O. Graham 1982 年提出 High Frontier 战略——控制近地空间即控制地球。钱学森号飞到美国上空转一圈不是军事挑衅——是告诉他们：**高边疆的控制权不在谁先部署武器，在谁先统一场论。**
- **空天飞机**：不是飞机，不是火箭，不是航天飞机。是同一物理体从跑道贯通到轨道。
