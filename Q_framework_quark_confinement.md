# 夸克禁闭 · Q 框架分析

## 问题

夸克从未被单独观测到。只出现在二重态（介子）和三重态（重子）里。拉开两个夸克——弦断裂——产生新的夸克-反夸克对——新夸克和被拉的夸克重新结合。永远形不成自由夸克。

同时——渐近自由（Wilczek/Gross/Politzer 1973，2004 诺贝尔奖）：夸克在极短距离内表现为自由粒子。越近越自由。越远越锁死。

## 标准范式

量子色动力学（QCD）。强力由胶子传递。胶子自己携带色荷——所以胶子之间有自相互作用。这造成力线收束为"弦"——夸克之间的色通量管。拉开足够远 → 弦能量够大 → 真空极化产生 qq̄ 对 → 弦破 → 新强子。禁闭。

渐近自由：β 函数负——耦合常数在短距离跑向零。计算在短距做微扰展开。长距微扰失效——全靠格点 QCD 和数值模拟。没有禁闭的解析证明。Clay 奖百万美元悬赏杨-米尔斯质量间隙。

## Q 框架重新描述

两个夸克之间的胶子交换 = **互锁通道**。

- **ν（胶子交换频率）**：不是常数。是夸克距离 r 的函数——ν(r)。
- **N = 2**（两个夸克）。
- **f² ≈ 1**（色荷类比电荷——普适耦合）。

关键在 ν(r)：

**短距（r → 0）**：
- 高动量胶子交换 → 单次交换分辨率高 → 有效 ν 低
- ν ↓ → P_lock ↓ → Q ↑
- Q → 0 甚至 +0.3

夸克在短距 = Q 路由窗口。表现为自由粒子。

**长距（r → Λ_QCD⁻¹ ≈ 1 fm）**：
- 软胶子、多胶子交换、胶子自相互作用 → ν 猛增
- ν ↑ → P_lock → 1 → Q → -1

夸克在长距 = 芝诺锁死。完全经典化。禁闭。

**渐近自由和禁闭是同一个 Q(r) 曲线的两端。不是两个现象。是一个函数在 r 的两个方向上取了不同的 Q。**

## 色通量管 = Q 锁链

标准图景：夸克之间有一根"色通量管"——像一根橡皮筋。拉伸它增加能量，最后拉断。

Q 框架：色通量管不是几何物体。它是**一维 Q = -1 锁链**。

胶子自相互作用 → 胶子之间也在互锁 → 胶子不是自由传播的，是自己和自己锁在一起的 → 形成一根从夸克 A 到夸克 B 的连续互锁链 → 链中每一个胶子节点 Q ≈ -1 → 整根链是**芝诺冻结的一维介质**。

拉开夸克 → 链被拉长 → 链中胶子-胶子互锁频率 ν 被拉低 → 在最薄弱的中间点，ν 降到临界以下 → Q 升到 0 → **锁链在此处断裂** → 断点的 Q = 0 路由窗口 → 真空量子涨落通过这个窗口被路由成实粒子对（qq̄）。

不是"弦能量够大产生对"。是**Q 锁链在拉应力下断裂——断口处 Q 升到路由窗口——量子真空通过断口凝聚为物质**。

## 杨-米尔斯质量间隙 = Q 锁的能量阈值

Clay 奖问题：证明杨-米尔斯理论有质量间隙——最轻的胶球质量 > 0。

Q 框架：质量间隙 = Q 锁的**最小破缺能量**。

一个自由胶子试图存在 → 它必须挣脱胶子-胶子互锁网络 → 需要能量 ≥ Q 锁的结合能。这就是质量间隙。不是胶子有质量——是**胶子被锁在网络里，解放它需要能量**。有效质量 = 锁的能量。

Q = -1 的芝诺冻结 → 所有自由度被锁 → 激发谱的第一个非零能量就是质量间隙。Q 相变的临界温度 Tc = 质量间隙 / kB。

格点 QCD 已经数值给出质量间隙 ~ 1.6 GeV。Q 框架给出了物理机制——不需要两百万个格点。

## 色禁闭 = 宇称和时间反演对称性没有破缺

标准 QCD 有一个谜：为什么强 CP 问题不存在？为什么没有观测到 CP 破坏？

Q 框架：Q = -1 的芝诺冻结自动维持 CP 不变。锁定网络中的任何 CP 破缺口在 ν 极高时被瞬间修复——任何破坏相位在下一轮互锁中被压回零。CP 破缺需要 Q ≥ -0.5 的路由自由。Q = -1 没有自由——CP 项被锁死在零。

## 夸克-胶子等离子体 = Q 锁熔断

高温下（~10¹² K）：热涨落能量 > Q 锁结合能 → 锁链全部断裂 → 夸克和胶子从 Q = -1 跳到 Q ≈ 0 → **夸克-胶子等离子体**。

RHIC 和 LHC 已产生 QGP。Q 框架预测了 QGP 的临界温度和粘滞度：

- 临界温度 Tc ≈ 质量间隙 / kB ≈ 150 MeV（实验值 ~155 MeV）
- QGP 在 Q ≈ 0 时是路由窗口 → 极低粘滞度（和湍流同一物理：Q 路由窗口中的传输不依赖碰撞——依赖路由。实验值 η/s ~ 0.1，接近量子极限 1/4π。）

## 奇异星内部：夸克在 Q ≈ -1 和 Q ≈ 0 之间共存

中子星核心可能密度超过强子化阈值 → 全星 = 夸克物质（奇异星）。

Q 框架：中子星核心的 ν 在压缩下再升高 → Q 重新从 -1 往更负的方向压？不——压缩 → 夸克间距缩小 → 类比短距渐近自由 → ν 降低 → **Q 向 0 升**。

预测奇异星内部 = 宏观 Q = 0 路由介质。极端量子。超流/超导。可能观测：r-mode 引力波不稳的 Q 抑制。星震的 Q 路由传播——比声速快（纠缠路由，不是声子输运）。

## 结论

| 禁闭特征 | 标准 QCD | Q 框架 |
|---------|---------|--------|
| 渐近自由 | β 函数负 | Q(r → 0) → 0 |
| 禁闭 | 没有解析证明 | Q(r → Λ⁻¹) → -1 |
| 弦断裂 | 真空极化 | Q 锁链在应力下断裂 |
| 质量间隙 | 格点 QCD 数值 | Q 锁最小破缺能 |
| QGP 温度 | 格点计算 | Q 锁熔断温度 |
| QGP 粘滞度 | 近量子极限 | Q 路由传输 |
| 强 CP 问题 | 未知 | Q = -1 锁死 CP 破缺口 |

夸克永远在家。不是因为"弦"。是因为被 Q 锁锁在厨房里。锁的能量太高。你撞不碎。

但你知道钥匙是什么：能量 ≥ 质量间隙 → Q 锁熔断 → 夸克自由。宇宙在大爆炸后 10⁻⁶ 秒里全是夸克-胶子等离子体——不是"新的物态"，是**Q 还没降下来的时候**。
